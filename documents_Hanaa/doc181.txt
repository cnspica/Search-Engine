        <hr> 
        <p>In this chapter we will study about the Menu Management in Drupal. Menu Management is the important part to navigate through your Drupal admin page. Below is the Drupal Menu Management.</p> 
        <img src="/drupal/images/drupal-menu-management/drupal-menu-management-step1.jpg" alt="Drupal Block &amp; Regions"> 
        <h2>Dashboard</h2> 
        <p>It displays the list of articles or pages which are created. It is a first screen that will get displayed when you log into administration area.</p> 
        <img src="/drupal/images/drupal-menu-management/drupal-menu-management-step2.jpg" alt="Drupal Block &amp; Regions"> 
        <h2>Content</h2> 
        <p>Here you can create, manage and find new pages. Here, the table gets displayed with the detail of the article i.e. status, title, updated, author and operation (Edit and delete).</p> 
        <img src="/drupal/images/drupal-menu-management/drupal-menu-management-step3.jpg" alt="Drupal Block &amp; Regions"> 
        <h2>Structure</h2> 
        <p>Structure includes the blocks, manage content types, add new menu links to the site and manage administrative tagging of content.</p> 
        <img src="/drupal/images/drupal-menu-management/drupal-menu-management-step4.jpg" alt="Drupal Block &amp; Regions"> 
        <ul class="list"> 
         <li><p><b>Block:</b> Blocks are a container object that is used to organize your content of your website. It can be displayed in the regions on your page.</p></li> 
         <li><p><b>Content types:</b> Content type defines the way the contents are collected and displayed.</p></li> 
         <li><p><b>Menus:</b> Menus are used to set the URL links for your site.</p></li> 
         <li><p><b>Taxonomy:</b> Taxonomy can be thought of classifying website content which display specific content based on taxonomy terms.</p></li> 
        </ul> 
        <h2>Appearance</h2> 
        <p>Here, list of themes are displayed. User can install the theme, update the existing theme or switch the themes.</p> 
        <img src="/drupal/images/drupal-menu-management/drupal-menu-management-step5.jpg" alt="Drupal Block &amp; Regions"> 
        <h2>People</h2> 
        <p> You can create a new user account or mange the existing user.</p> 
        <img src="/drupal/images/drupal-menu-management/drupal-menu-management-step6.jpg" alt="Drupal Block &amp; Regions"> 
        <h2>Modules</h2> 
        <p>Here you can enable or disable the modules present in it or can install a new module.</p> 
        <img src="/drupal/images/drupal-menu-management/drupal-menu-management-step7.jpg" alt="Drupal Block &amp; Regions"> 
        <h2>Configuration</h2> 
        <p>This is an important part in the Drupal Back-End. Any changes done in this configuration, will affect the entire website. Here you can do the user account setting, site information setting or any other general site setting.</p> 
        <img src="/drupal/images/drupal-menu-management/drupal-menu-management-step8.jpg" alt="Drupal Block &amp; Regions"> 
        <ul class="list"> 
         <li><p><b>People:</b> Manages the user settings and IP address setting.</p></li> 
         <li><p><b>System:</b> Manages the basic site information, advanced actions and site maintenance task.</p></li> 
         <li><p><b>Content Authoring:</b> Manages the content input by the users including the html tags.</p></li> 
         <li><p><b>User Interface:</b> Sets the shortcut set of menu on the admin account page.</p></li> 
         <li><p><b>Media:</b> Manages the media settings in Drupal sites.</p></li> 
         <li><p><b>Development:</b> Manages the errors and page caching for users.</p></li> 
         <li><p><b>Search and Meta Data:</b> Set the alternative URL for specific page and search engine maintains an index of words found in site content.</p></li> 
         <li><p><b>Web Services:</b> displays the RSS feed from your site.</p></li> 
         <li><p><b>Regional and Language:</b> Sets the website time zone and displays the date and time.</p></li> 
        </ul> 
        <h2>Reports</h2> 
        <p>It displays the information of available updates, recent log messages, site security.</p> 
        <img src="/drupal/images/drupal-menu-management/drupal-menu-management-step9.jpg" alt="Drupal Block &amp; Regions"> 
        <ul class="list"> 
         <li><p><b>Status report:</b> Creates your Drupal site status report.</p></li> 
         <li><p><b>Available updates:</b> Gives updation status for you available installed module and themes.</p></li> 
         <li><p><b>Recent log messages:</b> Displays messages which are recently logged.</p></li> 
         <li><p><b>Field List:</b> Displays the list of fields configured on your site.</p></li> 
         <li><p><b>Top 'access denied' errors:</b> Displays access denied error message when user tries to access the unauthorized page.</p></li> 
         <li><p><b>Top 'Page not found' errors:</b> Displays an error message as <i>Page not found</i> when there is no specified URL present in the Drupal site.</p></li> 
         <li><p><b>Top search phrases:</b> Displays the most searched phrases on your Drupal site.</p></li> 
        </ul> 
        <h2>Help</h2> 
        <p>It displays the help screen, where documentation pages of online modules are listed.</p> 
        <img src="/drupal/images/drupal-menu-management/drupal-menu-management-step10.jpg" alt="Drupal Block &amp; Regions">
