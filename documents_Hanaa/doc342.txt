        <h1>R - Factors</h1>

        <p>Factors are the data objects which are used to categorize the data and store it as levels. They can store both strings and integers. They are useful in the columns which have a limited number of unique values. Like "Male, "Female" and True, False etc. They are useful in data analysis for statistical modeling.</p> 
        <p>Factors are created using the <b>factor ()</b> function by taking a vector as input.</p> 
        <h2>Example</h2> 
        <pre class="prettyprint notranslate tryit">
# Create a vector as input.
data &lt;- c("East","West","East","North","North","East","West","West","West","East","North")

print(data)
print(is.factor(data))

# Apply the factor function.
factor_data &lt;- factor(data)

print(factor_data)
print(is.factor(factor_data))
</pre> 
        <p>When we execute the above code, it produces the following result −</p> 
        <pre class="result notranslate">
 [1] "East"  "West"  "East"  "North" "North" "East"  "West"  "West"  "West"  "East" "North"
[1] FALSE
 [1] East  West  East  North North East  West  West  West  East  North
Levels: East North West
[1] TRUE
</pre> 
        <h2>Factors in Data Frame</h2> 
        <p>On creating any data frame with a column of text data, R treats the text column as categorical data and creates factors on it.</p> 
        <pre class="prettyprint notranslate tryit">
# Create the vectors for data frame.
height &lt;- c(132,151,162,139,166,147,122)
weight &lt;- c(48,49,66,53,67,52,40)
gender &lt;- c("male","male","female","female","male","female","male")

# Create the data frame.
input_data &lt;- data.frame(height,weight,gender)
print(input_data)

# Test if the gender column is a factor.
print(is.factor(input_data$gender))

# Print the gender column so see the levels.
print(input_data$gender)
</pre> 
        <p>When we execute the above code, it produces the following result −</p> 
        <pre class="result notranslate">
  height weight gender
1    132     48   male
2    151     49   male
3    162     66 female
4    139     53 female
5    166     67   male
6    147     52 female
7    122     40   male
[1] TRUE
[1] male   male   female female male   female male  
Levels: female male
</pre> 
        <h2>Changing the Order of Levels</h2> 
        <p>The order of the levels in a factor can be changed by applying the factor function again with new order of the levels.</p> 
        <pre class="prettyprint notranslate tryit">
data &lt;- c("East","West","East","North","North","East","West","West","West","East","North")
# Create the factors
factor_data &lt;- factor(data)
print(factor_data)

# Apply the factor function with required order of the level.
new_order_data &lt;- factor(factor_data,levels = c("East","West","North"))
print(new_order_data)
</pre> 
        <p>When we execute the above code, it produces the following result −</p> 
        <pre class="result notranslate">
 [1] East  West  East  North North East  West  West  West  East  North
Levels: East North West
 [1] East  West  East  North North East  West  West  West  East  North
Levels: East West North
</pre> 
        <h2>Generating Factor Levels</h2> 
        <p>We can generate factor levels by using the <b>gl()</b> function. It takes two integers as input which indicates how many levels and how many times each level.</p> 
        <h3>Syntax</h3> 
        <pre class="result notranslate">
gl(n, k, labels)
</pre> 
        <p>Following is the description of the parameters used −</p> 
        <ul class="list"> 
         <li><p><b>n</b> is a integer giving the number of levels.</p></li> 
         <li><p><b>k</b> is a integer giving the number of replications.</p></li> 
         <li><p><b>labels</b> is a vector of labels for the resulting factor levels.</p></li> 
        </ul> 
        <h3>Example</h3> 
        <pre class="prettyprint notranslate tryit">
v &lt;- gl(3, 4, labels = c("Tampa", "Seattle","Boston"))
print(v)
</pre> 
        <p>When we execute the above code, it produces the following result −</p> 
        <pre class="result notranslate">
Tampa   Tampa   Tampa   Tampa   Seattle Seattle Seattle Seattle Boston 
[10] Boston  Boston  Boston 
Levels: Tampa Seattle Boston
</pre>
