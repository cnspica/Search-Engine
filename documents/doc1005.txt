        <p>Apache Maven is a software project management and comprehension tool. Based on the concept of a project object model (POM), Maven can manage a project's build, reporting and documentation from a central piece of information.</p> 
        <p>This tutorial will teach you how to use Maven in your day-2-day life of any project development using Java, or any other programming language.</p> 
        <h1>Audience</h1> 
        <p>This tutorial has been prepared for the beginners to help them understand basic functionality of Maven tool. After completing this tutorial you will find yourself at a moderate level of expertise in using Apache Maven from where you can take yourself to next levels.</p> 
        <h1>Prerequisites</h1> 
        <p>We assume you are going to use Maven to handle enterprise level Java projects development. So it will be good if you have knowledge of software development, Java SE, overview of Java EE development and deployment process.</p>
