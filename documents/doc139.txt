        <hr> 
        <p>In this chapter, we will study how to <b>create a template</b> in Joomla.</p> 
        <h2>Creating Templates</h2> 
        <p>Following are the simple steps to create templates in Joomla −</p> 
        <p><b>Step (1)</b> − Create a folder called <b>MyFirstTemplate</b> inside your <b>Joomla</b> → <b>Templates</b> folder. Inside <b>MyFirstTemplate</b> folder, create 2 more folders named as <i>images</i> and CSS to save all the images and CSS files.</p> 
        <img src="/joomla/images/joomla-createtemplates/joomla-createtemplates-step1.jpg" alt="Joomla Creating Template"> 
        <p><b>Step (2) −</b> Inside <b>MyFirstTemplate</b> folder, create a file called as <b>templateDetails.xml</b>, without this file the template will not get displayed in Joomla.</p> 
        <h2>templateDetails.xml</h2> 
        <pre class="prettyprint notranslate">
&lt;?xml version = "1.0" encoding = "utf-8"?&gt;

&lt;extension version = "3.0" type = "template"&gt;
   &lt;name&gt;Tutorials Point&lt;/name&gt;
   &lt;creationDate&gt;2015-06-13&lt;/creationDate&gt;
   &lt;author&gt;Tutorials Point&lt;/author&gt;
   &lt;authorEmail&gt;tutorials@example.com&lt;/authorEmail&gt;
   &lt;authorUrl&gt;http://www.example.com &lt;/authorUrl&gt;
   &lt;copyright&gt;Jack 2015&lt;/copyright&gt;
   &lt;license&gt;GNU/GPL&lt;/license&gt;
   &lt;version&gt;1.0.2&lt;/version&gt;
   &lt;description&gt;My First Template&lt;/description&gt;
	
   &lt;files&gt;
      &lt;filename&gt;index.php&lt;/filename&gt;
      &lt;filename&gt;templateDetails.xml&lt;/filename&gt;
      &lt;folder&gt;images&lt;/folder&gt;
      &lt;folder&gt;css&lt;/folder&gt;
   &lt;/files&gt;
	
   &lt;positions&gt;
      &lt;position&gt;breadcrumb&lt;/position&gt;
      &lt;position&gt;left&lt;/position&gt;
      &lt;position&gt;right&lt;/position&gt;
      &lt;position&gt;top&lt;/position&gt;
      &lt;position&gt;user1&lt;/position&gt;
      &lt;position&gt;user2&lt;/position&gt;
      &lt;position&gt;user3&lt;/position&gt;
     &lt;position&gt;user4&lt;/position&gt;
     &lt;position&gt;footer&lt;/position&gt;
   &lt;/positions&gt;
	
&lt;/extension&gt;
</pre> 
        <h3>Details of the code</h3> 
        <ul class="list"> 
         <li><p><b>&lt;files&gt;</b> − It contains the available files and folders in <b>MyFirstTemplate</b> folder.</p></li> 
         <li><p><b>&lt;folder&gt;</b> − It takes the entire available folders in <b>MyFirstTemplate</b> folder.</p></li> 
        </ul> 
        <p><b>Step (3)</b> − Create a file called as <b>index.php</b>. This helps in executing all the Joomla pages.</p> 
        <h2>index.php</h2> 
        <pre class="prettyprint notranslate">
&lt;?php defined( '_JEXEC' ) or die( 'Restricted access' );?&gt;
&lt;!DOCTYPE html&gt;

&lt;html xmlns = "http://www.w3.org/1999/xhtml" xml:lang = "&lt;?php echo $this-&gt;language; ?&gt;" lang = "&lt;?php echo $this-&gt;language; ?&gt;" &gt;

   &lt;head&gt;                                         //head section
      &lt;jdoc:include type = "head" /&gt;
      &lt;link rel = "stylesheet" href = "&lt;?php echo $this-&gt;baseurl ?&gt;/templates/system/css/system.css" type = "text/css" /&gt;
      &lt;link rel = "stylesheet" href = "&lt;?php echo $this-&gt;baseurl ?&gt;/templates/system/css/general.css" type = "text/css" /&gt;
      &lt;link rel = "stylesheet" href = "&lt;?php echo $this-&gt;baseurl ?&gt;/templates/&lt;?php echo $this-&gt;template; ?&gt;/css/template.css" type="text/css" /&gt;
   &lt;/head&gt;

   &lt;body&gt;                                          //body section
      &lt;jdoc:include type = "modules" name = "top" /&gt;
      &lt;jdoc:include type = "component" /&gt;
      &lt;jdoc:include type = "modules" name = "bottom" /&gt;
   &lt;/body&gt;
	
&lt;/html&gt;

?&gt;
</pre> 
        <h3>Details of the code</h3> 
        <pre class="prettyprint notranslate">
&lt;?php defined( '_JEXEC' ) or die( 'Restricted access' );?&gt;
&lt;!DOCTYPE html&gt;
&lt;html xmlns = "http://www.w3.org/1999/xhtml" xml:lang = "&lt;?php echo $this-&gt;language; ?&gt;" lang = "&lt;?php echo $this-&gt;language; ?&gt;" &gt;
</pre> 
        <p>This code is used to tell the browser which type of html page is being used and begins the HTML document by describing the language used in the website.</p> 
        <pre class="prettyprint notranslate">
&lt;head&gt;                                         //head section
   &lt;jdoc:include type = "head" /&gt;
   &lt;link rel = "stylesheet" href = "&lt;?php echo $this-&gt;baseurl ?&gt;/templates/system/css/system.css" type = "text/css" /&gt;
   &lt;link rel = "stylesheet" href = "&lt;?php echo $this-&gt;baseurl ?&gt;/templates/system/css/general.css" type = "text/css" /&gt;
   &lt;link rel = "stylesheet" href = "&lt;?php echo $this-&gt;baseurl ?&gt;/templates/&lt;?php echo $this-&gt;template; ?&gt;/css/template.css" type = "text/css" /&gt;
&lt;/head&gt;
</pre> 
        <p>These lines are used to link the various stylesheets for the template in Joomla.</p> 
        <pre class="prettyprint notranslate">
&lt;body&gt;                                          //body section
   &lt;jdoc:include type = "modules" name = "top" /&gt;
   &lt;jdoc:include type = "component" /&gt;
   &lt;jdoc:include type = "modules" name = "bottom" /&gt;
&lt;/body&gt;
</pre> 
        <p>Here in the body section, the <b>jdoc</b> is used to include the output in Joomla from some parts of Joomla system. The <b>name = "top"</b> is used to set the menu at the top.</p> 
        <p><b>Step (4)</b> − After creating these two files, login to Joomla and click on <b>Extension</b> → <b>Extension Manager</b> and the following page will get displayed.</p> 
        <img src="/joomla/images/joomla-createtemplates/joomla-createtemplates-step4.jpg" alt="Joomla Creating Template"> 
        <p><b>Step (5)</b> − In the above page, click on <b>Discover</b> link, the following page is displayed. </p> 
        <img src="/joomla/images/joomla-createtemplates/joomla-createtemplates-step5.jpg" alt="Joomla Creating Template"> 
        <p><b>Step (6)</b> − Next, click on the <b>Discover</b> button to discover the new created template as shown below.</p> 
        <img src="/joomla/images/joomla-createtemplates/joomla-createtemplates-step6.jpg" alt="Joomla Creating Template"> 
        <p><b>Step (7)</b> − Click on the checkbox to select the template and click on <b>Install</b> button to install the template in Joomla.</p> 
        <img src="/joomla/images/joomla-createtemplates/joomla-createtemplates-step7.jpg" alt="Joomla Creating Template"> 
        <p><b>Step (8)</b> − After installing template you can view your new created template by clicking on <b>Extension</b> → <b>Template Manager</b>. You will see your new created template is listed as shown below.</p> 
        <img src="/joomla/images/joomla-createtemplates/joomla-createtemplates-step8.jpg" alt="Joomla Creating Template"> 
        <h2>Toolbar</h2> 
        <p>Following are the toolbar options in the Template Manager −</p> 
        <ul class="list"> 
         <li><p><b>Default</b> − Select the default template.</p></li> 
         <li><p><b>Edit</b> − Select the template for editing.</p></li> 
         <li><p><b>Duplicate</b> − Duplicate the copy of the selected template.</p></li> 
         <li><p><b>Delete</b> − Delete the template for Joomla.</p></li> 
        </ul>
