        <p>This tutorial describes various ways to incorporate <b>JFreeChart</b> in Java based standalone and web based applications. It will give you a quick start with JFreeChart and make you comfortable with JFreeChart programming with Java applications.</p> 
        <h1>Audience</h1> 
        <p>This reference has been prepared for the beginners to help them understand the basic to advanced concepts related to JFreeChart library.</p> 
        <h1>Prerequisites</h1> 
        <p>Before you start practising with various code examples given in the tutorial, I assume you are already aware of Java programming with JDK1.6 or later versions, swing, file management and database management in Java.</p>
