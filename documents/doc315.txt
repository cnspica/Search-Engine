        <p>Lists are the R objects which contain elements of different types like − numbers, strings, vectors and another list inside it. A list can also contain a matrix or a function as its elements. List is created using <b>list()</b> function.</p> 
        <h2>Creating a List</h2> 
        <p>Following is an example to create a list containing strings, numbers, vectors and a logical values</p> 
        <pre class="prettyprint notranslate tryit">
# Create a list containing strings, numbers, vectors and a logical values.
list_data &lt;- list("Red", "Green", c(21,32,11), TRUE, 51.23, 119.1)
print(list_data)
</pre> 
        <p>When we execute the above code, it produces the following result −</p> 
        <pre class="result notranslate">
[[1]]
[1] "Red"

[[2]]
[1] "Green"

[[3]]
[1] 21 32 11

[[4]]
[1] TRUE

[[5]]
[1] 51.23

[[6]]
[1] 119.1
</pre> 
        <h2>Naming List Elements</h2> 
        <p>The list elements can be given names and they can be accessed using these names.</p> 
        <pre class="prettyprint notranslate tryit">
# Create a list containing a vector, a matrix and a list.
list_data &lt;- list(c("Jan","Feb","Mar"), matrix(c(3,9,5,1,-2,8), nrow = 2),
   list("green",12.3))

# Give names to the elements in the list.
names(list_data) &lt;- c("1st Quarter", "A_Matrix", "A Inner list")

# Show the list.
print(list_data)
</pre> 
        <p>When we execute the above code, it produces the following result −</p> 
        <pre class="result notranslate">
$`1st_Quarter`
[1] "Jan" "Feb" "Mar"

$A_Matrix
     [,1] [,2] [,3]
[1,]    3    5   -2
[2,]    9    1    8

$A_Inner_list
$A_Inner_list[[1]]
[1] "green"

$A_Inner_list[[2]]
[1] 12.3
</pre> 
        <h2>Accessing List Elements</h2> 
        <p>Elements of the list can be accessed by the index of the element in the list. In case of named lists it can also be accessed using the names.</p> 
        <p>We continue to use the list in the above example −</p> 
        <pre class="prettyprint notranslate tryit">
# Create a list containing a vector, a matrix and a list.
list_data &lt;- list(c("Jan","Feb","Mar"), matrix(c(3,9,5,1,-2,8), nrow = 2),
   list("green",12.3))

# Give names to the elements in the list.
names(list_data) &lt;- c("1st Quarter", "A_Matrix", "A Inner list")

# Access the first element of the list.
print(list_data[1])

# Access the thrid element. As it is also a list, all its elements will be printed.
print(list_data[3])

# Access the list element using the name of the element.
print(list_data$A_Matrix)
</pre> 
        <p>When we execute the above code, it produces the following result −</p> 
        <pre class="result notranslate">
$`1st_Quarter`
[1] "Jan" "Feb" "Mar"

$A_Inner_list
$A_Inner_list[[1]]
[1] "green"

$A_Inner_list[[2]]
[1] 12.3

     [,1] [,2] [,3]
[1,]    3    5   -2
[2,]    9    1    8
</pre> 
        <h2>Manipulating List Elements</h2> 
        <p>We can add, delete and update list elements as shown below. We can add and delete elements only at the end of a list. But we can update any element.</p> 
        <pre class="prettyprint notranslate tryit">
# Create a list containing a vector, a matrix and a list.
list_data &lt;- list(c("Jan","Feb","Mar"), matrix(c(3,9,5,1,-2,8), nrow = 2),
   list("green",12.3))

# Give names to the elements in the list.
names(list_data) &lt;- c("1st Quarter", "A_Matrix", "A Inner list")

# Add element at the end of the list.
list_data[4] &lt;- "New element"
print(list_data[4])

# Remove the last element.
list_data[4] &lt;- NULL

# Print the 4th Element.
print(list_data[4])

# Update the 3rd Element.
list_data[3] &lt;- "updated element"
print(list_data[3])
</pre> 
        <p>When we execute the above code, it produces the following result −</p> 
        <pre class="result notranslate">
[[1]]
[1] "New element"

$
NULL

$`A Inner list`
[1] "updated element"
</pre> 
        <h2>Merging Lists</h2> 
        <p>You can merge many lists into one list by placing all the lists inside one list() function.</p> 
        <pre class="prettyprint notranslate tryit">
# Create two lists.
list1 &lt;- list(1,2,3)
list2 &lt;- list("Sun","Mon","Tue")

# Merge the two lists.
merged.list &lt;- c(list1,list2)

# Print the merged list.
print(merged.list)
</pre> 
        <p>When we execute the above code, it produces the following result −</p> 
        <pre class="result notranslate">
[[1]]
[1] 1

[[2]]
[1] 2

[[3]]
[1] 3

[[4]]
[1] "Sun"

[[5]]
[1] "Mon"

[[6]]
[1] "Tue"
</pre> 
        <h2>Converting List to Vector</h2> 
        <p>A list can be converted to a vector so that the elements of the vector can be used for further manipulation. All the arithmetic operations on vectors can be applied after the list is converted into vectors. To do this conversion, we use the <b>unlist()</b> function. It takes the list as input and produces a vector.</p> 
        <pre class="prettyprint notranslate tryit">
# Create lists.
list1 &lt;- list(1:5)
print(list1)

list2 &lt;-list(10:14)
print(list2)

# Convert the lists to vectors.
v1 &lt;- unlist(list1)
v2 &lt;- unlist(list2)

print(v1)
print(v2)

# Now add the vectors
result &lt;- v1+v2
print(result)
</pre> 
        <p>When we execute the above code, it produces the following result −</p> 
        <pre class="result notranslate">
[[1]]
[1] 1 2 3 4 5

[[1]]
[1] 10 11 12 13 14

[1] 1 2 3 4 5
[1] 10 11 12 13 14
[1] 11 13 15 17 19
</pre>
