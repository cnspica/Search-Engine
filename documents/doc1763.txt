             <p>Staightforward with a stack.</p> 
             <pre><code>\documentclass{article}
\usepackage{stackengine,graphicx,trimclip,scalerel}
\begin{document}
\savestack\eye{\rotatebox{90}{$^\circ\mkern-6mu\raisebox{1pt}{)}$}}
\savestack\nose{\raisebox{3pt}{\scalebox{1}[-1]{\clipbox{0pt 1pt 0pt 0pt}{?}}}}
\savestack\mouth{\rotatebox{90}{(}}
\scaleleftright{(}{\stackanchor[2pt]{\eye \nose \eye}{\mouth}}{)}
\end{document}
</code></pre> 
             <p><a href="http://i.stack.imgur.com/TSPDw.jpg"><img src="http://i.stack.imgur.com/TSPDw.jpg" alt="enter image description here"></a></p> 
             <p>From there to a macro follows directly:</p> 
             <pre><code>\documentclass{article}
\usepackage{stackengine,graphicx,trimclip,scalerel}
\savestack\eye{\rotatebox{90}{$^\circ\mkern-6mu\raisebox{1pt}{)}$}}
\savestack\nose{\raisebox{3pt}{\scalebox{1}[-1]{\clipbox{0pt 1pt 0pt 0pt}{?}}}}
\savestack\mouth{\rotatebox{90}{(}}
\newcommand\Lenny{\scaleleftright{(}{\stackanchor[2pt]{\eye \nose \eye}{\mouth}}{)}}
\begin{document}
This is my Lenny face: \Lenny
\end{document}
</code></pre> 
             <p><a href="http://i.stack.imgur.com/3BrVY.jpg"><img src="http://i.stack.imgur.com/3BrVY.jpg" alt="enter image description here"></a></p> 
             <p>And here is a variant that won't distort line spacing:</p> 
             <pre><code>\documentclass{article}
\usepackage[nopar]{lipsum}
\usepackage{stackengine,graphicx,trimclip,scalerel}
\savestack\eye{\rotatebox{90}{$^\circ\mkern-6mu\raisebox{1pt}{)}$}}
\savestack\nose{\raisebox{3pt}{\scalebox{1}[-1]{\clipbox{0pt 1pt 0pt 0pt}{?}}}}
\savestack\mouth{\rotatebox{90}{(}}
\newcommand\Lenny{(\scalerel{\stackanchor[2pt]{\eye \nose \eye}{\mouth}}{)}}
\begin{document}
\lipsum[2] \Lenny{} \lipsum[3]
\end{document}
</code></pre> 
             <p><a href="http://i.stack.imgur.com/4tu95.jpg"><img src="http://i.stack.imgur.com/4tu95.jpg" alt="enter image description here"></a></p> 
             <hr> 
             <p>Finally, a more proportioned variation, that can present Lenny from different font families (by way of optional argument):</p> 
             <pre><code>\documentclass{article}
\usepackage[nopar]{lipsum}
\usepackage{stackengine,graphicx,trimclip,scalerel}
\newcommand\Lenny[1][]{\bgroup#1%
  \savestack\eye{\rotatebox{90}{\footnotesize$^\circ\mkern-6mu\raisebox{1pt}{)}$}}%
  \savestack\nose{\raisebox{4pt}{\scalebox{1}[-1.3]{\clipbox{0pt 1.5pt 0pt 0pt}{?}}}}%
  \savestack\mouth{\rotatebox{90}{(}}%
  (\kern-1pt\scalerel{\stackanchor[1.0pt]{$\eye\nose\!\eye$}{\mouth}}{\kern-1pt)}%
  \egroup%
}
\begin{document}
{\tiny rmfamily sffamily mirrored-ttfamily}\par
\Lenny\quad
\Lenny[\sffamily]\quad
\scalebox{-1}[1]{\Lenny[\ttfamily]}
\end{document}
</code></pre> 
             <p><a href="http://i.stack.imgur.com/Hr2NA.jpg"><img src="http://i.stack.imgur.com/Hr2NA.jpg" alt="enter image description here"></a></p>

             <p>You can use the following code (based on an example by David Griffiths):</p> 
             <pre><code>\documentclass{report}
\usepackage{graphicx}
\def\len{{\mbox{$\resizebox{.31in}{.08in}{\includegraphics{len}}$}}}
\begin{document}

    Here's a sample:
    $\resizebox{.31in}{.08in}{\includegraphics{len}}$

    Can I put it into a line of type?  $\resizebox{.43in}{.11in}{\includegraphics{len}}$
    How about using the macro: \len

    \begin{equation}
        1 + \len = 0
    \end{equation}
\end{document}
</code></pre>
