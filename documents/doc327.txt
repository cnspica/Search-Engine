        <p>Ruby is a scripting language designed by Yukihiro Matsumoto, also known as Matz. It runs on a variety of platforms, such as Windows, Mac OS, and the various versions of UNIX. This tutorial gives a complete understanding on Ruby.</p> 
        <h1>Audience</h1> 
        <p>This reference has been prepared for the beginners to help them understand the basic to advanced concepts related to Ruby Scripting languages.</p> 
        <h1>Prerequisites</h1> 
        <p>Before you start doing practice with various types of examples given in this reference, I'm making an assumption that you are already aware about what is a computer program and what is a computer programming language.</p> 
        <h1>Execute Ruby Online</h1> 
        <p>For most of the examples given in this tutorial you will find Try it option, so just make use of this option to execute your Ruby programs at the spot and enjoy your learning.</p> 
        <p>Try following example using Try it option available at the top right corner of the below sample code box −</p>

#!/usr/bin/ruby -w

puts "Hello, Ruby!";
