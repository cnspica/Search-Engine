            <div class="post-text" itemprop="text"> 
             <p>This question is related to <a href="http://tex.stackexchange.com/questions/268871/carry-out-substring-replacement-on-variables-contents">Carry out substring replacement on variable's contents</a>. Is it possible to perform string replace on an arbitrary string, without expansion? MWE:</p> 
             <pre><code>\documentclass{article}
\usepackage{xstring}
\begin{document}
\noexpandarg
\StrSubstitute{John \textit{knows} that Mary is a unicorn}{knows}{believes}
\end{document}
</code></pre> 
             <p>With this MWE, nothing is replaced. Desired behaviour is a replacement of <code>\textit{knows}</code> with <code>\textit{believes}</code>, i.e., verbatim, plain replacement. Also, e.g., <code>\StrSubstitute{\textit{knows}}{tex}{latex}</code> should result in <code>\latexit{knows}</code> (i.e., replacement should consider the string as given, without taking any internal semantics into consideration).</p> 
             <p>A <code>pdflatex</code> solution is preferred (no lua, perl, sed, text editor search and replace). <code>xstring</code> is not required.</p>

            <div class="post-text" itemprop="text"> 
             <p>By default xstring doesn't look into groups. But you can force it to consider also the text in braces:</p> 
             <pre><code>\documentclass{article}
\usepackage{xstring}
\begin{document}
\noexpandarg
\StrSubstitute{John {knows} that Mary is a unicorn}{knows}{believes}

\exploregroups
\StrSubstitute{John {knows} that Mary is a unicorn}{knows}{believes}

\StrSubstitute{John \textit{knows} that Mary is a unicorn}{knows}{believes}
\end{document} 
</code></pre>

             <p>Such a simple substitution can be obtained quite easily with <code>l3regex</code>.</p> 
             <pre><code>\documentclass{article}
\usepackage{xparse,l3regex}

\ExplSyntaxOn
\NewDocumentCommand{\strsubstitute}{ +m m m }
 {
  \tl_set:Nn \l_marijn_strsubst_tl { #1 }
  \regex_replace_all:nnN { #2 } { #3 } \l_marijn_strsubst_tl
  \tl_use:N \l_marijn_strsubst_tl
 }
\tl_new:N \l_marijn_strsubst_tl
\ExplSyntaxOff

\begin{document}

\strsubstitute{John \textit{knows} that Mary is a unicorn}{knows}{believes}

\strsubstitute{John \textit{knows} that Mary is a unicorn}{\c{textit}}{\c{textbf}}

\end{document}
</code></pre>
