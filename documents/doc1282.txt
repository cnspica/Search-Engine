        <h1>R - CSV Files</h1>

        <hr> 
        <p>In R, we can read data from files stored outside the R environment. We can also write data into files which will be stored and accessed by the operating system. R can read and write into various file formats like csv, excel, xml etc.</p> 
        <p>In this chapter we will learn to read data from a csv file and then write data into a csv file. The file should be present in current working directory so that R can read it. Of course we can also set our own directory and read files from there.</p> 
        <h2>Getting and Setting the Working Directory</h2> 
        <p>You can check which directory the R workspace is pointing to using the <b>getwd()</b> function. You can also set a new working directory using <b>setwd()</b>function.</p> 
        <pre class="prettyprint notranslate tryit">
# Get and print current working directory.
print(getwd())

# Set current working directory.
setwd("/web/com")

# Get and print current working directory.
print(getwd())
</pre> 
        <p>When we execute the above code, it produces the following result −</p> 
        <pre class="result notranslate">
[1] "/web/com/1441086124_2016"
[1] "/web/com"
</pre> 
        <p>This result depends on your OS and your current directory where you are working.</p> 
        <h2>Input as CSV File</h2> 
        <p>The csv file is a text file in which the values in the columns are separated by a comma. Let's consider the following data present in the file named <b>input.csv</b>.</p> 
        <p>You can create this file using windows notepad by copying and pasting this data. Save the file as <b>input.csv</b> using the save As All files(*.*) option in notepad.</p> 
        <pre class="prettyprint notranslate" id="input_csv">
id,name,salary,start_date,dept
1,Rick,623.3,2012-01-01,IT
2,Dan,515.2,2013-09-23,Operations
3,Michelle,611,2014-11-15,IT
4,Ryan,729,2014-05-11,HR
 ,Gary,843.25,2015-03-27,Finance
6,Nina,578,2013-05-21,IT
7,Simon,632.8,2013-07-30,Operations
8,Guru,722.5,2014-06-17,Finance
</pre> 
        <h2>Reading a CSV File</h2> 
        <p>Following is a simple example of <b>read.csv()</b> function to read a CSV file available in your current working directory −</p> 
        <pre class="prettyprint notranslate tryit" title=",,,input_csv">
data &lt;- read.csv("input.csv")
print(data)
</pre> 
        <p>When we execute the above code, it produces the following result −</p> 
        <pre class="result notranslate">
      id,   name,    salary,   start_date,     dept
1      1    Rick     623.30    2012-01-01      IT
2      2    Dan      515.20    2013-09-23      Operations
3      3    Michelle 611.00    2014-11-15      IT
4      4    Ryan     729.00    2014-05-11      HR
5     NA    Gary     843.25    2015-03-27      Finance
6      6    Nina     578.00    2013-05-21      IT
7      7    Simon    632.80    2013-07-30      Operations
8      8    Guru     722.50    2014-06-17      Finance
</pre> 
        <h2>Analyzing the CSV File</h2> 
        <p>By default the <b>read.csv()</b> function gives the output as a data frame. This can be easily checked as follows. Also we can check the number of columns and rows.</p> 
        <pre class="prettyprint notranslate tryit" title=",,,input_csv">
data &lt;- read.csv("input.csv")

print(is.data.frame(data))
print(ncol(data))
print(nrow(data))
</pre> 
        <p>When we execute the above code, it produces the following result −</p> 
        <pre class="result notranslate">
[1] TRUE
[1] 5
[1] 8
</pre> 
        <p>Once we read data in a data frame, we can apply all the functions applicable to data frames as explained in subsequent section.</p> 
        <h3>Get the maximum salary</h3> 
        <pre class="prettyprint notranslate tryit" title=",,,input_csv">
# Create a data frame.
data &lt;- read.csv("input.csv")

# Get the max salary from data frame.
sal &lt;- max(data$salary)
print(sal)
</pre> 
        <p>When we execute the above code, it produces the following result −</p> 
        <pre class="result notranslate">
[1] 843.25
</pre> 
        <h3>Get the details of the person with max salary</h3> 
        <p>We can fetch rows meeting specific filter criteria similar to a SQL where clause.</p> 
        <pre class="prettyprint notranslate tryit" title=",,,input_csv">
# Create a data frame.
data &lt;- read.csv("input.csv")

# Get the max salary from data frame.
sal &lt;- max(data$salary)

# Get the person detail having max salary.
retval &lt;- subset(data, salary == max(salary))
print(retval)
</pre> 
        <p>When we execute the above code, it produces the following result −</p> 
        <pre class="result notranslate">
      id    name  salary  start_date    dept
5     NA    Gary  843.25  2015-03-27    Finance
</pre> 
        <h3>Get all the people working in IT department</h3> 
        <pre class="prettyprint notranslate tryit" title=",,,input_csv">
# Create a data frame.
data &lt;- read.csv("input.csv")

retval &lt;- subset( data, dept == "IT")
print(retval)
</pre> 
        <p>When we execute the above code, it produces the following result −</p> 
        <pre class="result notranslate">
       id   name      salary   start_date   dept
1      1    Rick      623.3    2012-01-01   IT
3      3    Michelle  611.0    2014-11-15   IT
6      6    Nina      578.0    2013-05-21   IT
</pre> 
        <h3>Get the persons in IT department whose salary is greater than 600</h3> 
        <pre class="prettyprint notranslate tryit" title=",,,input_csv">
# Create a data frame.
data &lt;- read.csv("input.csv")

info &lt;- subset(data, salary &gt; 600 &amp; dept == "IT")
print(info)
</pre> 
        <p>When we execute the above code, it produces the following result −</p> 
        <pre class="result notranslate">
       id   name      salary   start_date   dept
1      1    Rick      623.3    2012-01-01   IT
3      3    Michelle  611.0    2014-11-15   IT
</pre> 
        <h3>Get the people who joined on or after 2014</h3> 
        <pre class="prettyprint notranslate tryit" title=",,,input_csv">
# Create a data frame.
data &lt;- read.csv("input.csv")

retval &lt;- subset(data, as.Date(start_date) &gt; as.Date("2014-01-01"))
print(retval)
</pre> 
        <p>When we execute the above code, it produces the following result −</p> 
        <pre class="result notranslate">
       id   name     salary   start_date    dept
3      3    Michelle 611.00   2014-11-15    IT
4      4    Ryan     729.00   2014-05-11    HR
5     NA    Gary     843.25   2015-03-27    Finance
8      8    Guru     722.50   2014-06-17    Finance
</pre> 
        <h2>Writing into a CSV File</h2> 
        <p>R can create csv file form existing data frame. The <b>write.csv()</b> function is used to create the csv file. This file gets created in the working directory.</p> 
        <pre class="prettyprint notranslate tryit" title=",,,input_csv">
# Create a data frame.
data &lt;- read.csv("input.csv")
retval &lt;- subset(data, as.Date(start_date) &gt; as.Date("2014-01-01"))

# Write filtered data into a new file.
write.csv(retval,"output.csv")
newdata &lt;- read.csv("output.csv")
print(newdata)
</pre> 
        <p>When we execute the above code, it produces the following result −</p> 
        <pre class="result notranslate">
  X      id   name      salary   start_date    dept
1 3      3    Michelle  611.00   2014-11-15    IT
2 4      4    Ryan      729.00   2014-05-11    HR
3 5     NA    Gary      843.25   2015-03-27    Finance
4 8      8    Guru      722.50   2014-06-17    Finance
</pre> 
        <p>Here the column X comes from the data set newper. This can be dropped using additional parameters while writing the file.</p> 
        <pre class="prettyprint notranslate tryit" title=",,,input_csv">
# Create a data frame.
data &lt;- read.csv("input.csv")
retval &lt;- subset(data, as.Date(start_date) &gt; as.Date("2014-01-01"))

# Write filtered data into a new file.
write.csv(retval,"output.csv", row.names = FALSE)
newdata &lt;- read.csv("output.csv")
print(newdata)
</pre> 
        <p>When we execute the above code, it produces the following result −</p> 
        <pre class="result notranslate">
      id    name      salary   start_date    dept
1      3    Michelle  611.00   2014-11-15    IT
2      4    Ryan      729.00   2014-05-11    HR
3     NA    Gary      843.25   2015-03-27    Finance
4      8    Guru      722.50   2014-06-17    Finance
</pre>
