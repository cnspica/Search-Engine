         <p>An aptitude test represents a sytematic way to judge a candidate's mental capability as how he/she performs certain tasks and reacts to different scenarios. Each aptitude test uses a standard method of administration and scoring so that results can be computed and compared with the other candidates.</p> 
         <p>This tutorial will teach you basic concepts of aptitude like numerical computation ability, analytical abilities. The tutorial is divided into sections such as Number System, Basic Arithmetic, Decimals and so on. Each of these sections contain related topics with simple and useful examples.</p>

        <h1>Audience</h1> 
        <p>This reference has been prepared for the candidates to help them to understand the basic to advanced concepts required to pass the aptitude tests. This tutorial will give you enough understanding on various concepts where you can take yourself to a higher level of expertise.</p>
