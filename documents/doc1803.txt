    &lt;h3&gt;&nbsp;&lt;span style="font-weight: 400"&gt;How do I place a bootable ISO on a USB drive?&lt;/span&gt;&lt;/h3&gt; &lt;p&gt;&lt;span style="font-weight: 400"&gt;(&lt;/span&gt;&lt;a href="https://superuser.com/questions/66948/how-do-i-place-a-bootable-iso-on-a-usb-drive"&gt;&lt;span style="font-weight: 400"&gt;Question ID 66948&lt;/span&gt;&lt;/a&gt;&lt;span style="font-weight: 400"&gt;)&lt;/span&gt;&lt;/p&gt; &lt;p&gt;This is a popular question with many similar or overlapping questions. Browsing the various linked and related questions I noticed the information on this subject is limited and fragmented. Most answers are simple software recommendations and do not touch upon any underlying issues. This article is meant to help fellow superusers appreciate the assumptions we take for granted and serve as a springboard for those that need to dig deeper to solve their problem.&lt;/p&gt; &lt;p&gt;&lt;span style="font-weight: 400"&gt;Most people who are looking to place an ISO on a USB drive want to&nbsp;&lt;/span&gt;&lt;span style="font-weight: 400"&gt;put &lt;/span&gt;&lt;i&gt;&lt;span style="font-weight: 400"&gt;any&lt;/span&gt;&lt;/i&gt;&lt;span style="font-weight: 400"&gt; ISO on a USB drive and make the drive bootable (a&lt;/span&gt;lso known as making a live USB or installer USB from a binary image file). Examples include:&lt;/p&gt; &lt;ul&gt; &lt;li&gt;Windows installer / Windows recovery&lt;/li&gt; &lt;li&gt;Some other&nbsp;rescue media&nbsp;needed to restore a&nbsp;backup&nbsp;(e.g. &lt;a href="http://www.sevenforums.com/tutorials/218920-macrium-reflect-create-bootable-rescue-usb-drive.html"&gt;Macrium reflect&lt;/a&gt;)&lt;/li&gt; &lt;li&gt;Linux live image, possibly &lt;a href="https://www.maketecheasier.com/persistent-live-usb-vs-full-install-usb/"&gt;persistent&lt;/a&gt;&lt;/li&gt; &lt;li&gt;A full OS install onto a USB&lt;/li&gt; &lt;li&gt;does &lt;strong&gt;not&lt;/strong&gt; include useless&nbsp;ISOs,&nbsp;e.g. non-data like those filled with music or video.&lt;/li&gt; &lt;/ul&gt; &lt;p&gt;&lt;span style="font-weight: 400"&gt;This is &lt;/span&gt;&lt;b&gt;&lt;i&gt;not &lt;/i&gt;&lt;/b&gt;&lt;span style="font-weight: 400"&gt;necessarily&nbsp;the same as:&lt;/span&gt;&lt;/p&gt; &lt;ul&gt; &lt;li style="font-weight: 400"&gt;&lt;span style="font-weight: 400"&gt;putting an ISO on a bootable USB drive &lt;/span&gt;&lt;/li&gt; &lt;li style="font-weight: 400"&gt;&lt;span style="font-weight: 400"&gt;copying an ISO file or its filesystem contents onto a formatted USB drive&lt;/span&gt;&lt;/li&gt; &lt;/ul&gt; &lt;p&gt;&lt;span style="font-weight: 400"&gt;The problem we solve with today&amp;#8217;s post is: &amp;#8220;Given an ISO and a non-bootable USB drive, how do I create a bootable USB?”&lt;/span&gt;&lt;/p&gt; &lt;h3&gt;&lt;strong&gt;What is an ISO anyway?&lt;/strong&gt;&lt;/h3&gt; &lt;p&gt;&lt;span style="font-weight: 400"&gt;In the broadest sense it is a binary image (a.k.a. carbon copy, snapshot or raw data backup) of the storage sectors of a block-device such as a DVD, USB drive, micro-sd card or HDD. The ISO format has no standard (ironically), so there is no pre-defined header (if any), nor any restriction on what filesystems may be contained. These image files are &lt;/span&gt;&lt;i&gt;&lt;span style="font-weight: 400"&gt;not &lt;/span&gt;&lt;/i&gt;&lt;span style="font-weight: 400"&gt;related to graphical/visual image files like JPEGs. &lt;em&gt;&lt;strong&gt;But&lt;/strong&gt; &lt;/em&gt;in common usage an ISO is expected to consist of a single standard filesystem, like the&nbsp;ISO 9660 filesystem.&nbsp;&lt;/span&gt;&lt;span style="font-weight: 400"&gt;Here is another good definition (from &lt;a href="http://www.magiciso.com/tutorials/miso-whatiso.htm"&gt;MagicISO &lt;/a&gt;website):&nbsp;&lt;/span&gt;&lt;/p&gt; &lt;blockquote&gt;Going by the more restrictive definition, an &amp;#8220;ISO&amp;#8221; is created by copying an entire disc, from sector 0&nbsp;to the end, into a file. [&amp;#8230;]&nbsp;it isn&amp;#8217;t possible to store anything but a single data track in this fashion. Audio tracks, mixed-mode discs, CD+G, multisession, and other fancy formats can&amp;#8217;t be represented. [this specifically refers to discs with an ISO 9660 filesystem, as a&nbsp;&lt;em&gt;convention&lt;/em&gt;]&lt;/blockquote&gt; &lt;p&gt;In case you&amp;#8217;re wondering about tracks, they&amp;#8217;re basically partitions&amp;#8211;so obviously a single filesystem can not have partitions. Of course an image can contain multiple tracks/partitions, but if we refer to such an image as an ISO we are using the loose definition (and probably making some developer angry). So an ISO can be any data image, but by &lt;strong&gt;convention&lt;/strong&gt; we mean the entire&nbsp;user data* from an optical disk, which is commonly formatted with a &lt;strong&gt;single filesystem&lt;/strong&gt;. So your ISO will &lt;em&gt;probably&lt;/em&gt; conform to this more restrictive definition.&lt;/p&gt; &lt;p&gt;&lt;span style="font-weight: 400"&gt;The name originated from CD images that commonly used ISO 9660 to store data. Nowadays most uses assume an ISO refers to an image from an optical disk like CD or DVD (even though in DVD the UDF filesystem is more common) . Even &lt;a href="http://wiki.osdev.org/Mkisofs"&gt;tools&lt;/a&gt;&nbsp;that can create an ISO from a HDD folder probably implicitly convert to an optical media filesystem like ISO 9660. For this reason it may be better for software to use a different extension like “.IMG” to denote non-optical image files.&lt;/span&gt;&lt;/p&gt; &lt;p&gt;*With CDs a raw sector is 2352 bytes, but only 2048 bytes are used for data in formatted storage. The rest is used for error correction related stuff. This is why you might see corrupted files if you extract a true raw image from a CD and try to mount it&amp;#8211;files are injected with junk bits. DVDs don&amp;#8217;t make this distinction, their sectors only have 2048 bytes of user data. So in the case of CDs with filesystems an ISO is normally not a true sector-by-sector carbon copy.&lt;/p&gt; &lt;blockquote&gt;These days it&amp;#8217;s not altogether uncommon to see messages about &amp;#8220;making an ISO&amp;#8221; of an audio CD, which makes no sense at all. (&lt;a href="http://www.magiciso.com/tutorials/miso-whatiso.htm"&gt;MagicISO FAQ&lt;/a&gt;)&lt;/blockquote&gt; &lt;h3&gt;&lt;strong&gt;Aha, a copy &amp;#8216;sectors&amp;#8217; from a &amp;#8216;block device&amp;#8217;&amp;#8230;&lt;/strong&gt;&lt;/h3&gt; &lt;p&gt;A&nbsp;block device is just jargon for storage media, or &amp;#8216;drive&amp;#8217; if you like. From a programmer&amp;#8217;s point of view storage is a one dimensional array (row/line) of bits. But at the hardware level you can&amp;#8217;t address each bit individually. The smallest &amp;#8216;word&amp;#8217; size for reading and writing is called the sector size. Filesystems are implemented to improve aspects like speed or security (e.g. we&amp;#8217;d like for&nbsp;files to have names instead of a sector number like 0x0000008AF1). It has been a while since I attempted graphics:&lt;/p&gt; &lt;div id="attachment_5634" style="width: 693px" class="wp-caption alignleft"&gt;&lt;a href="http://blog.superuser.com/files/2016/01/block_device3.png"&gt;&lt;img class="wp-image-5634 size-full" src="http://blog.superuser.com/files/2016/01/block_device3.png" alt="test" width="683" height="204" /&gt;&lt;/a&gt;&lt;p class="wp-caption-text"&gt;Figure 1 Block Device: The smallest addressable unit in a block device is called a sector. CDs and DVDs typically have a sector size of 2048 bytes. Filesystems (managed by the OS) may group multiple sectors into blocks. E.g. 4 sectors in the physical addressing space may constitute 1 block address in the logical (filesystem) addressing space&amp;#8211;see image. The same data may occupy different sectors depending on the filesystem type. In the case of ISO 9660 a block is normally also 2048 bytes (but this value isn&amp;#8217;t demanded by&nbsp;the standard).&lt;/p&gt;&lt;/div&gt; &lt;p&gt;So files occupy &lt;em&gt;at least&lt;/em&gt;&nbsp;1 sector. Images (ISOs in the loose definition) are created by copying all the bits in all the sectors. In case the data occupies less than the whole sector (e.g small files)&nbsp;the other bits are still copied instead of being zeroed out, because the filesystem may require them. On a sidenote, in my opinion file based images (like WIM used in Windows backups) do not fit the definition of &amp;#8220;image&amp;#8221;.&nbsp;But that is a discussion for another article.&lt;/p&gt; &lt;p&gt;The size of a sector depends on the hardware. Careful when burning an image your software does not make any wrong assumptions. For example copying every two original sectors into one new one might mess up the filesystem.&lt;/p&gt; &lt;blockquote&gt;Beginning in late 2009, accelerating in 2010 and hitting mainstream in 2011, hard drive companies are migrating away from the legacy sector size of 512 bytes to a larger, more efficient sector size of 4096 bytes, generally referred to as 4K sectors, and now referred to as the Advanced Format by IDEMA (The International Disk Drive Equipment and Materials Association). (&lt;a href="http://www.seagate.com/gb/en/tech-insights/advanced-format-4k-sector-hard-drives-master-ti/"&gt;seagate&lt;/a&gt;)&lt;/blockquote&gt; &lt;div id="attachment_5648" style="width: 310px" class="wp-caption alignleft"&gt;&lt;a href="http://blog.superuser.com/files/2016/01/manage_images.png"&gt;&lt;img class="size-medium wp-image-5648" src="http://blog.superuser.com/files/2016/01/manage_images-300x143.png" alt="You can manage disk images with tools like these (bit outdated)." width="300" height="143" /&gt;&lt;/a&gt;&lt;p class="wp-caption-text"&gt;You can manage disk images with tools like these.&lt;/p&gt;&lt;/div&gt; &lt;p&gt;&lt;span style="font-weight: 400"&gt;Tools that can create an ISO from optical media are common. Most disk burning software will also have this functionality. Though it seems much harder to find software that will make a true image of non-optical media (like a HDD or USB drive). Most backup software (Acronis, Paragon, Norton, etc.) &nbsp;will do this, but they may use proprietary formats and/or compression.&nbsp;&lt;/span&gt;&lt;/p&gt; &lt;p&gt;&lt;span style="font-weight: 400"&gt;It is also possible to image individual partitions and even files on a hard drive. Mounting or file exploring software will need to recognize the filesystem before it becomes usable. Ok so, enough about images, how do we&nbsp;use&nbsp;some&nbsp;ISO with a&nbsp;flash drive?&lt;/span&gt;&lt;/p&gt; &lt;blockquote&gt;*A sidenote for Windows users: If you want to quickly get an ISO from your DVD or explore the files of one, 7zip might be all you need. If you want to make an ISO from folders you’ll need to try something more advanced like imgburn&amp;#8211;it lets you add/load a custom bootsector. Surprisingly few utilities support making ISOs from folders.&nbsp;Note that only adding a bootsector will not re-create the original ISO if hidden settings were lost.&lt;/blockquote&gt; &lt;p&gt;&lt;strong&gt;Bootable vs Non-bootable&lt;/strong&gt;&lt;/p&gt; &lt;p&gt;&lt;span style="font-weight: 400"&gt;The question as it is now specifically refers to bootable ISOs as opposed to non-bootable ISOs. Is there such a distinction, and if so, what’s the difference? Well, bootable ISOs contain ‘invisible’ data called &nbsp;boot-sectors. Now this is where it gets messy because many people don&amp;#8217;t know what kind of firmware they have (BIOS or UEFI) and it determines the bootcode requirements.&lt;/span&gt;&lt;/p&gt; &lt;p&gt;&lt;em&gt;UEFI&lt;/em&gt;&lt;/p&gt; &lt;p&gt;&lt;span style="font-weight: 400"&gt;UEFI systems don&amp;#8217;t need this invisible data to boot, provided your files include .efi files in the &lt;a href="https://www.thomas-krenn.com/en/wiki/Creating_Windows_UEFI_Boot-Stick_in_Windows"&gt;right place&lt;/a&gt; on your USB drive. The drive should also be marked active and be formatted with the FAT32 filesystem. Simple. This is why some cases of copy and pasting the ISO contents just work, while others do the same steps and fail miserably on their BIOS systems. In certain&nbsp;cases the USB drive has an MBR instead of a GPT and the UEFI goes into BIOS mode without trying to look for the .efi files. To fix this bad behavior either change a setting in the UEFI or change the drive to a GPT type partitioning.&lt;/span&gt;&lt;/p&gt; &lt;p&gt;&lt;em&gt;BIOS&lt;/em&gt;&lt;/p&gt; &lt;p&gt;So technically an ISO with efi files might be bootable, but I will reserve the term &amp;#8220;bootable ISO&amp;#8221; for those images with invisible boot files. For the BIOS case you will need this invisible data (i.e. a bootable ISO) unless your USB drive was previously formatted with compatible boot code, in which case you can just replace the visible filesystem contents from the ISO&amp;#8211;copy paste.&nbsp;Additionally, in some rare cases the BIOS might be&nbsp;smart enough to boot without a sector like UEFI.&lt;/p&gt; &lt;p&gt;&lt;span style="font-weight: 400"&gt;There are normally two sectors involved in loading the operating system, the&nbsp;&lt;a href="https://en.wikipedia.org/wiki/Master_boot_record"&gt;MBR &lt;/a&gt;and the &lt;a href="https://en.wikipedia.org/wiki/Volume_boot_record"&gt;VBR&lt;/a&gt;. This implies a&nbsp;BIOS system can&amp;#8217;t&nbsp;boot from a GPT style drive. To get this invisible data from your ISO to your USB you need to burn the image, not just copy the filesystem (visible) files. If you don&amp;#8217;t know if your ISO is bootable, burn it and test it, or maybe check the &lt;a href="https://superuser.com/questions/683210/how-do-i-determine-if-an-iso-is-a-hybrid"&gt;first sector&lt;/a&gt;.&lt;/span&gt;&lt;/p&gt; &lt;blockquote&gt;“ The location and size of the boot sector (perhaps corresponding to a logical disk sector) is specified by the design of the computing platform.”&lt;/blockquote&gt; &lt;p&gt;&lt;span style="font-weight: 400"&gt;&amp;#8212; Wikipedia (&lt;/span&gt;&lt;a href="https://en.wikipedia.org/wiki/Boot_sector"&gt;&lt;span style="font-weight: 400"&gt;Boot_sector&lt;/span&gt;&lt;/a&gt;&lt;span style="font-weight: 400"&gt;)&lt;/span&gt;&lt;/p&gt; &lt;p&gt;&lt;span style="font-weight: 400"&gt;So it’s a few Kb somewhere that the CPU loads to do the bootstrapping process. You can&amp;#8217;t access this data with a file explorer, because&nbsp;it is&nbsp;not part of the filesystem proper. A good ISO image editor should let you modify&nbsp;all the bytes, possibly overwriting them with custom sub-images.&lt;/span&gt;&lt;/p&gt; &lt;p&gt;&lt;span style="font-weight: 400"&gt;Now the basic theory is out of the way, perhaps&nbsp;a better question is: what about all those free tools that take an ISO and give back a bootable USB?&nbsp;I suppose some tools may&nbsp;add the invisible stuff for you if you have&nbsp;a non-bootable ISO. Or they might choose between MBR and GPT for you. Or you might be having trouble finding software you like that burns your ISO to your drive. They just make life a bit easier. But they might not consider all the&nbsp;variables I&amp;#8217;ve mentioned.&lt;/span&gt;&lt;/p&gt; &lt;p&gt;&lt;span style="font-weight: 400"&gt;The most important thing to remember is a USB drive can be made bootable independently of the data. The following general steps can be taken to create our live/installer USB:&lt;/span&gt;&lt;/p&gt; &lt;ol&gt; &lt;li&gt;If you don&amp;#8217;t know if your ISO is bootable, burn and test it. If that fails then wipe and:&lt;/li&gt; &lt;li&gt;&lt;span style="font-weight: 400"&gt;Make USB bootable by adding bootsector(s), formatting may wipe the drive. A common tool is &lt;a href="https://technet.microsoft.com/en-us/library/cc749177(v=ws.10).aspx"&gt;bootsect.exe&lt;/a&gt;.&nbsp;This is needed for BIOS systems. This process (particularly the VBR) is often OS specific, but many non-windows solutions use GRUB or Syslinux (&lt;a href="https://wiki.archlinux.org/index.php/Boot_loaders"&gt;link&lt;/a&gt;). There are many tutorials on the web about how to install one of these bootsectors. But before you get stuck in, invest some more time into finding a working image file, most developers distribute them.&nbsp;&lt;/span&gt;&lt;/li&gt; &lt;li&gt;&lt;span style="font-weight: 400"&gt;Add &lt;/span&gt;&lt;i&gt;&lt;span style="font-weight: 400"&gt;compatible&lt;/span&gt;&lt;/i&gt;&lt;span style="font-weight: 400"&gt; files (e.g. linux distro, windows installer, rescue program). You can just copy the files contained by the ISO to your USB drive. Depending on your target operating system your folder structure may need to be in&nbsp;a specific format. In &lt;a href="http://www.pendrivelinux.com/install-grub2-on-usb-from-ubuntu-linux/"&gt;some cases&lt;/a&gt; you are allowed to copy the whole ISO.&lt;/span&gt;&lt;/li&gt; &lt;/ol&gt; &lt;p&gt;&lt;span style="font-weight: 400"&gt;No-one wants to burn a DVD twice, even if they could. But once you have a bootable USB drive you can simply copy the data or folders to it. As long as your invisible boot-sector is valid for whatever you want to use you can just swap files. I believe Vista and up use compatible boot code for BIOS and UEFI. An alternative route is to modify the ISO directly and then burn it all at once.&nbsp;&lt;/span&gt;&lt;/p&gt; &lt;p&gt;&lt;span style="font-weight: 400"&gt;&nbsp;Though we agreed to follow the ISO convention, ISO hybrids exist. They&nbsp;already contain the MBR code the flash drive needs while also keeping the 9660 filesystem intact. Rarely are USB-only images distributed (unfortunately).&lt;/span&gt;&lt;/p&gt; &lt;div id="attachment_5635" style="width: 310px" class="wp-caption alignnone"&gt;&lt;a href="http://blog.superuser.com/files/2016/01/optical_disk_layout.png"&gt;&lt;img class="size-medium wp-image-5635" src="http://blog.superuser.com/files/2016/01/optical_disk_layout-300x140.png" alt="The above shows how a possible hybrid ISO can host both types of bootsector (optical and HDD). Only the green area is viewable to a typical user when mounting the partition." width="300" height="140" /&gt;&lt;/a&gt;&lt;p class="wp-caption-text"&gt;The above shows how a possible hybrid ISO can host both types of bootsector (optical and HDD). Only the green area is viewable to a typical user when mounting the partition.&lt;/p&gt;&lt;/div&gt; &lt;p&gt;&lt;span style="font-weight: 400"&gt;There are many tools like the official Microsoft tool (probably uses bootsect), Rufus, YUMI (a.k.a pendrivelinux) and Unetbootin. I&amp;#8217;m not sure how these programs do everything, but&nbsp;I believe most software will require a bootable ISO to generate the bootable USB sector. Perhaps they translate the boot volume descriptor (optical disk filesystem) into an MBR compatible with whatever filesystem they use to format the drive (probably FAT32, or maybe they leave it in 9660).&lt;/span&gt;&lt;/p&gt; &lt;p&gt;&lt;b&gt;So what if you have a bootable ISO, but it has&nbsp;become unbootable?&lt;/b&gt;&lt;/p&gt; &lt;p&gt;&lt;span style="font-weight: 400"&gt;Normally non-bootable ISOs are useless for the purposes of live/installer USBs (e.g. a DVD video ISO)&amp;#8211;unless you somehow accidentally end up with a non-bootable ISO that &lt;em&gt;should&lt;/em&gt; have been bootable.&lt;/span&gt;&lt;/p&gt; &lt;p&gt;&lt;span style="font-weight: 400"&gt;For example, if you take a bootable DVD you can create an ISO from it both with or without the bootable property, depending on how you do it. In fact you can save any file or folder in ISO format, because there is no standard&amp;#8211;it is just a filesystem snapshot. It is the settings used in the ISO creation process that determine if the produced ISO is bootable or not. &lt;/span&gt;&lt;/p&gt; &lt;p&gt;&lt;span style="font-weight: 400"&gt;People often end up with non-bootable ISOs when they try to customize their ISO or create an ISO from a folder. The bootable property is lost when extracting an ISO and then repacking it without setting ‘make bootable’ options. Be sure to copy your bootsector.bin from the original ISO.&lt;/span&gt;&lt;/p&gt; &lt;p&gt;&lt;span style="font-weight: 400"&gt;Now, let’s say you have a folder with files you want to turn into a bootable USB. You can either manually make the USB bootable, but this may be tricky for some&nbsp;images/systems. Or you can try and turn the folder into a bootable ISO, which you then supply to some one-click tool to create the live/installer USB (you’d need to find/create a bootsector first&amp;#8211;and no guarantee it will work). Lastly there may be a niche tool that converts your customized folder directly to a working official ISO (like WAIK or&nbsp;RT se7en Lite for Windows images).&lt;/span&gt;&lt;/p&gt; &lt;p&gt;&lt;span style="font-weight: 400"&gt;Note the lack of 2&nbsp;options: there doesn’t seem to be software that accepts a folder directly, probably due to the lack of bootsector info, though it should work for modern UEFI systems. And there doesn’t seem to be a tool that can make a USB bootable for a given image distribution (i.e. just do step 2&nbsp;in a few&nbsp;clicks). I&amp;#8217;d be happy to hear information to the contrary!&lt;/span&gt;&lt;/p&gt; &lt;h3&gt;&lt;strong&gt;Conclusion&lt;/strong&gt;&lt;/h3&gt; &lt;p&gt;&lt;span style="font-weight: 400"&gt;Many one-click tools do require an ISO (preferably a bootable one) to generate a bootable USB. Just remember that an ISO may be&nbsp;more than just a file archive, and the underlying filesystems may vary. Get the right ISO and any burning software will copy all the sectors to your USB drive, it shouldn&amp;#8217;t be hard. And don&amp;#8217;t be worried&nbsp;if the extension is something like .img or .bin, in many cases an image is an image!&lt;/span&gt;&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt;

    These days, most PCs come with sound cards that are good enough for most purposes. Speakers have gotten generic enough that shopping for a good pair, with something to differentiate them, isn’t that easy. In a sense, if you aren&amp;#8217;t massively picky, you aren&amp;#8217;t going to &nbsp;look very hard or far, since what you have [&amp;#8230;]
   </description> 
   <content:encoded>
    &lt;p&gt;These days, most PCs come with sound cards that are good enough for most purposes. Speakers have gotten generic enough that shopping for a good pair, with something to differentiate them, isn’t that easy. In a sense, if you aren&amp;#8217;t massively picky, you aren&amp;#8217;t going to &nbsp;look very hard or far, since what you have is probably good enough.&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;h2&gt;&lt;b&gt;Output devices&lt;/b&gt;&lt;/h2&gt; &lt;p&gt;I&amp;#8217;d start by saying that sound is subjective &amp;#8211; different people expect different things from their sound system, and look at different things. A pretty good idea when actually trying to put together gear for sound output is to decide what you&amp;#8217;re going to do, and what you expect from your sound gear. Different folk expect different things &amp;#8211; some value bass over all else (in which case, you&amp;#8217;d want a pretty powerful subwoofer, or bass oriented headphones), while other want a neutral sound.&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;An ideal sound system sounds exactly like what its source material sounds like. Practically, its pretty hard to make a device that records the entire range of human hearing (ideally from 20 &amp;#8211; 20,000 hz). Your mic locations and types make a difference. The recorded material might also be changed in storage (for example, if it’s converted to an MP3), and by the output devices. In some cases, distortion can be pleasing (as vinyl and tube enthusiasts will tell you). In short, an &lt;i&gt;ideal &lt;/i&gt;sound playback system is impossible. At the end of the day, rather than perfect fidelity, a realistic goal is simply a system you are happy with.&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;In most cases, unless you&amp;#8217;re on the loony fringe of audiophilia, the component closest to your ears is probably the most crucial part of your setup. It’s going to determine what you have behind it and whether you get the most out of it. (I doubt you can get very good sound from a home-made monstrosity of tin cans, cellophane and magnet wire ripped out from alternators, even with the world&amp;#8217;s best amplifier behind it!)&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;At the most basic level, everyone has speakers somewhere these days &amp;#8211; most modern monitors have a tiny pair hidden away somewhere, as do laptops. One step above that would be the good old stereo speakers. These are perfectly adequate for transforming an electrical signal into sound. I use a pair of old Altec Lansings for gaming and system sounds, but I’d do just as well with my monitor’s built in speakers in many cases. These speakers probably are museum pieces- they possibly predate front panel audio, but they&amp;#8217;re solid, well made and sound surprisingly good.&lt;/p&gt; &lt;p&gt;&lt;a href="http://blog.superuser.com/files/2014/06/IMG_1071-Mobile.jpg"&gt;&lt;img class="alignleft size-medium wp-image-5575" src="http://blog.superuser.com/files/2014/06/IMG_1071-Mobile-300x224.jpg" alt="IMG_1071 (Mobile)" width="300" height="224" /&gt;&lt;/a&gt;&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;If you need more bass, a subwoofer, as part of a 2.1 speaker set would give you that rumble; however, if you have small animals at home, be warned it may frighten them. Hardcore gamers might look at surround sound speakers &amp;#8211; ones with more speakers such as 5.1 sets split sound into more channels you can position for more immersive sound.&lt;/p&gt; &lt;p&gt;&lt;a href="http://blog.superuser.com/files/2014/06/21.jpg"&gt;&lt;img class="alignleft size-medium wp-image-5576" src="http://blog.superuser.com/files/2014/06/21-300x189.jpg" alt="21" width="300" height="189" /&gt;&lt;/a&gt;&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;Monitors are a step up &nbsp;in terms of sound quality- these are full range speakers which are often used to monitor (hence the name) recordings. They often offer a full range of frequencies (that’s to say they cover the ranges subwoofers do as well &amp;#8211; not all speakers do), and are significantly better quality than most regular PC speakers in most respects. They also tend to be larger, but not absurdly large, and contain their own internal amplifiers.&nbsp; They do tend to be slightly harder to find(As with a lot of my audio gear, I find that music shops rather than computer shops are the best place to find these), but are a great option if you want good sound in small spaces.&lt;/p&gt; &lt;p&gt;&lt;a href="http://blog.superuser.com/files/2014/06/SPav40_front.jpg"&gt;&lt;img class="alignleft size-medium wp-image-5577" src="http://blog.superuser.com/files/2014/06/SPav40_front-300x194.jpg" alt="SPav40_front" width="300" height="194" /&gt;&lt;/a&gt;&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;Of course, there’s nothing stopping you from plugging in your PC to an amplifier or receiver, and running a regular pair of speakers off it. Lots of people use their PCs as you would any other source with great results.&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;Personally, I favour headphones. They’re generally cheaper (with some exceptions), some of them offer decent or even spectacular sound quality and, are good if you’re primarily listening to music or gaming alone. There’s a vast array of different types, but there’s two main things to be aware of.&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;Firstly the ‘form factor’ of the headphones &amp;#8211; ranging from simple earbuds, to In Ear Monitors or IEMs, which go into the ear canal to earphones that fit on your ear to circumaural (headphones that sit around your ear) and circumaural (headphones that sit on your ear). The second is the design of the phones, such as the driver design. IEMs can be dynamic, or use balanced armatures, headphones are mostly dynamic, but there’s alternate designs with their own fans. Headphones can be open-backed (which sound more natural, but often have less bass, and leak noise) or closed back (better isolation, more bass). Quite often, listening to a pair of headphones is the only way to be sure!&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;In some cases your speakers and headphones may be too demanding for your onboard sound card to drive, or have the wrong interfaces, so it’s useful to know what you have. Many monitors may take in 1/4 inch inputs or RCA, which your regular sound card may not have, and some headphones need a more powerful output than your regular sound card can handle. A good rule of thumb here would be the impedance level, and that most soundcards can handle up to 32 Ω (ohm) headphones with no issues. Impedance is a hugely complicated thing, and for our purposes, just note that you just don’t want to run a high impedance headphone off a regular sound card.&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;In the next part, we’ll talk about options for feeding your output devices, and when a regular sound card just doesn&amp;#8217;t cut it.&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;h2&gt;Part 2: Feeding your speakers or headphones the good stuff&lt;/h2&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;As with your output devices, different people expect different things from their sources. By a ‘source’, I really mean “device that feeds sound into your sound system so you hear things” &amp;#8211; traditionally this could have been a DVD player or even some flavour of a tape player. Associated with that would be the amplification stage of things.&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;In the context of what you’re likely to run into, we can really classify 3 types of sources.&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;Firstly, the traditional sound card and your integrated sound devices. Integrated sound devices usually are designed around providing the bare minimum of outputs at the lowest cost. While there’s some motherboards with fancy features, in most cases you’d have 5 channel sound and S/PDIF. While soundcards are similar, since basic features have been commodified; these often offer better headphone amplification stages, more options, and ostensibly better sound chips. If you’re a gamer, soundcards are likely the best option, since they’re built around gaming needs, such as surround sound. Creative seem to have gotten their drivers in order these days and make really nice sound cards, including models that work with recent phones, and Asus Xonar is pretty well known.&nbsp; You&amp;#8217;re likely going to need a spare PCI or PCIe (Most modern sound cards use a PCIe 1x connector) slot for an internal card, or spare USB port. While typically these are one slot card, there are occationally 2 slot cards with additiona features &amp;#8211; such as this creative zxr&lt;/p&gt; &lt;p&gt;&lt;a href="http://blog.superuser.com/files/2014/06/zxr.jpg"&gt;&lt;img class="alignleft size-medium wp-image-5578" src="http://blog.superuser.com/files/2014/06/zxr-300x240.jpg" alt="zxr" width="300" height="240" /&gt;&lt;/a&gt;&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;The second is the DAC or digital audio converter. These are units used by audiophiles which usually do purely output and either have a headphone out, or can be connected to an amplifier. These tend to be somewhat expensive, you’ll need to do your research but in many cases should offer the best sound quality for music. These are usually connected to your PC via USB and are detected as a soundcard. These tend to be expensive (though prices have been dropping), and as with any audiophile grade equipment, you can’t always trust reviews. Do &lt;i&gt;lots&lt;/i&gt; of research, and check out as many reviews you can. If you can, many shops that sell this sort of &nbsp;gear have display models you can audition. These are occationally homebrew units or kits, though there&amp;#8217;s pre-built ones with great build quality. I&amp;#8217;d note though, some companies make a hash of perfectly good sound chips so do your homework before you invest in one of these. These are in effect &lt;em&gt;very&lt;/em&gt; fancy soundcards.&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;The third is the DAW or Digital Audio Workstation &amp;#8211; these are basically DAC-type devices with better inputs &amp;#8211; I use one of these for playback, and if you do your homework, it offers good sound quality, often at lower prices. &nbsp;They’re pretty good if you want to use proper mics, or want something that can output to RCA &amp;#8211; my music playing setup is built around a relatively inexpensive, rather old M Audio Fast Track, so I obviously believe this is a good option. Its got a decent headphone output, RCA out if you want to connect to an amplifier or monitors, and a proper mic and guitar/direct input&lt;/p&gt; &lt;p&gt;&lt;a href="http://blog.superuser.com/files/2014/06/rtaImage.jpg"&gt;&lt;img class="alignleft size-medium wp-image-5579" src="http://blog.superuser.com/files/2014/06/rtaImage-300x200.jpg" alt="rtaImage" width="300" height="200" /&gt;&lt;/a&gt;&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;There’s devices that straddle lines &amp;#8211; there’s headphone amplifiers that work as a DAC/microphone for phones for example, or dedicated amplifiers and so on &amp;#8211; so I may not cover the whole spectrum of things you may come across&lt;/p&gt; &lt;p&gt;As with your output devices, what you have might be perfectly adequate, and there’s usually no need for fancy, extremely expensive audiophile gear in most cases. I found that the most sensible reason for a sound device outside your onboard sound card was really your inputs and outputs. I got a DAW cause I needed proper XLR microphone ports for music (and it turned out, it sounded better than my onboard sound card for music).&lt;/p&gt; &lt;p&gt;I’ve chosen to have an amplifier between my source and my headphones. This is pretty much optional, but if you have more demanding headphones or want to use speakers not actually designed for a PC, it can be worthwhile upgrade. It’s useful to know what inputs your amplifier takes in (mine does 3.5mm and RCA) and what your soundcard does. In general, you’re &lt;i&gt;not&lt;/i&gt; going to need an amplifier except for very special circumstances, and you will know when that is &amp;#8211; high impedance headphones, and regular unpowered speakers come to mind. While there&amp;#8217;s a slight improvement to sound quality in my opinion, the big difference is in convenience &amp;#8211; I have two inputs for my amplifier so I can switch between my PC (where I game) and my laptop with a flick of a switch, and have a large, physical volume control.&nbsp; Its a sAPII made by a&nbsp; chinese company called SMSL which I bought on a whim.&lt;/p&gt; &lt;p&gt;&lt;a href="http://blog.superuser.com/files/2014/06/Untitled.jpg"&gt;&lt;img class="alignleft size-medium wp-image-5580" src="http://blog.superuser.com/files/2014/06/Untitled-300x227.jpg" alt="Untitled" width="300" height="227" /&gt;&lt;/a&gt;&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;Finally of course, you need to get sound to your system &amp;#8211; games would do it pretty naturally, but there’s a few tips and tricks for music I use, which I’ll cover in the concluding part.&lt;/p&gt; &lt;h2&gt;&lt;/h2&gt; &lt;h2&gt;Part 3: If music is the food of love (of music?), play on!&lt;/h2&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;From parts 1 and 2, it’s rather obvious that I believe that in most cases, your standard sound setup is good enough. I &lt;i&gt;am&lt;/i&gt; rather picky about music however, and try to get the best, least-coloured sound out of my system as possible. I also want my music player to stay out of my way.&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;Like a lot of audiophiles, I’m a fan of &lt;a href="http://www.foobar2000.org"&gt;foobar2000&lt;/a&gt; &amp;#8211; there’s alternatives of various sorts such as lilith on windows but I find that a nearly stock foobar2000 setup works best for me.&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;I do choose to use bit-perfect playback where possible &amp;#8211; originally through ASIO, and later with WASAPI (there’s a third option of kernel streaming). All these methods bypass the kernel mixer which I find, subjectively, improves sound quality &amp;#8211; (&lt;a href="&lt;a href="http://wiki.hydrogenaudio.org/index.php?title=Bypassing_Windows_Mixer"&gt;http://wiki.hydrogenaudio.org/index.php?title=Bypassing_Windows_Mixer&lt;/a&gt;&amp;#8220;&gt;Hydrogen Audio disagrees&lt;/a&gt; though: if some driver uses bad compression and EQ presets, the difference from bit perfect playback can be dramatic). Once again, I find experimenting to be the best option here &amp;#8211; you may not hear a difference here. Since I use seperate sound devices for music and other stuff, I simply set my DAW into WASAPI exclusive mode, so that gets a stream of music, while the onboard sound card handles other sounds.&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;At some point, tweaking your software is a little like trying to overclock &amp;#8211; you might get some improvement, real or perceived, but it isn’t going to make a major difference if you’re working off hardware that’s not really very good.&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;I also favour lossless &amp;#8211; &lt;a href="&lt;a href="http://www.codinghorror.com/blog/2012/06/concluding-the-great-mp3-bitrate-experiment.html"&gt;http://www.codinghorror.com/blog/2012/06/concluding-the-great-mp3-bitrate-experiment.html&lt;/a&gt;&amp;#8220;&gt;Jeff Atwood disagrees&lt;/a&gt;, but these days, storage space is cheap. I have the option of re-converting my audio with no quality loss. While &lt;a href="http://www.dbpoweramp.com"&gt;dbpoweramp&lt;/a&gt; is what I hear rave reviews about as far as ripping CDs go, I favour &lt;a href="&lt;a href="http://www.cuetools.net/wiki/Main_Page"&gt;http://www.cuetools.net/wiki/Main_Page&lt;/a&gt;&amp;#8220;&gt;cuetools&lt;/a&gt; &amp;#8211; it supports some of the same features, is free &nbsp;and does GPU accelerated ripping.&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;Once again, your mileage may vary. Lots of people swear by &lt;a href="http://www.winamp.com"&gt;Winamp&lt;/a&gt;, or just listen to music off &lt;a href="https://www.youtube.com"&gt;YouTube&lt;/a&gt; or &lt;a href="https://www.spotify.com"&gt;Spotify&lt;/a&gt;. It’s &lt;i&gt;pretty&lt;/i&gt; unlikely you’ll need a fancy setup for that. Built up to your needs, and your preferences. I personally pretty much accidentally discovered my preferences, and bought my initial audiophile gear and upgraded as things broke or I wanted to experiment. &nbsp;Nearly my whole setup consists of iterations of settings and gear until it sounded &lt;i&gt;right&lt;/i&gt;. I’ve made one or two mistakes of course, but the current iteration sounds pretty good, and handles what I throw at it. I don’t tweak my sound for gaming &amp;#8211; I’ve not seen much options, and if you’re familiar with how to tweak a computer for better gaming sound, why not write an article for the Super User Blog!&lt;/p&gt;

    &lt;p&gt;Buying a new laptop can be a difficult venture. You must decide which one is right for you. Depending on what your needs are, there are tons of things to consider like hard drive space, graphics cards, and general ease of use.&lt;/p&gt; &lt;p&gt;But once you find your dream computer, there are a few things that you must do. Here are some tips to follow after you purchase your laptop to make your computing experience a pleasurable one.&lt;/p&gt; &lt;h2&gt;Register and Update Windows&lt;/h2&gt; &lt;p&gt;&lt;a href="http://blog.superuser.com/files/2013/10/win8.jpg"&gt;&lt;img class="alignnone size-full wp-image-5527" alt="win8" src="http://blog.superuser.com/files/2013/10/win8.jpg" width="500" height="333" /&gt;&lt;/a&gt;&lt;/p&gt; &lt;p&gt;Image via &lt;a href="http://www.flickr.com/photos/46411239@N05/8121841101/in/photolist-dnGyp8-dnHY2R-dnHG8k-d2HaGf-d2H6Fs-d2H8Ry-d2H8CW-d2Hb4u-bmCLbh-doAybA-dTaUm9-eguBC6-brXLxq-bESDEM-bESDux-bzphKr-apQp58-apQp56-bVmug4-bAL7w3-dnU4xL-92andR-doAxzQ-doAtnn-cz65YA-doAri2-doApue-doABxd-doAvTq-dovQmo-dovGSp-dovCvv-dcvNUn-bPEM8T-amWZJK-amX12v-cqpQ21-dihuNQ-f7Da7N-f7oYoX-f7oYCD-dP5mLe-dGi7eA-dorutS-dnDuwT-em732U-dsSyjE-bnhnaL-ciT1ZE-e6UyZ8-brXLU5" target="_blank"&gt;Flickr&lt;/a&gt; by Microsoft Sweden&lt;/p&gt; &lt;p&gt;An important part of purchasing your computer is actually registering (and &lt;a href="http://windows.microsoft.com/en-us/windows-8/why-activate-windows"&gt;successfully activating&lt;/a&gt;) Windows. It activates all the perks of having Windows as an operating system, such as Windows Media Player, and it also enables desktop personalization.&lt;/p&gt; &lt;p&gt;Next you’ll want to download all system updates and service packs. You’ll want a really fast &lt;a href="http://www.dsl.com/tools/internet-cable-breakdown/"&gt;internet connection&lt;/a&gt; for this, because these can be huge files and take a while to download. However, they’re vital to making your computer safer and run much smoother. These updates patch up any bugs or glitches that were newly found, and they streamline the performance of the operating system as well as add new features. &nbsp;To &lt;a href="http://superuser.com/questions/545115/is-windows-7-home-better-for-small-notebook-than-professional"&gt;answer super user “Hennes” question&lt;/a&gt;, it doesn’t matter which variation of Windows 7 you have, Pro or Home, it will run, performance wise, the same, if you’re worried about performance issues think about either adding more RAM or upgrading to Windows 8, which for the for the most part is a more streamlined and smoother operating system.&lt;/p&gt; &lt;h2&gt;Rid your Computer from Unwanted “Bloatware”&lt;/h2&gt; &lt;p&gt;When you purchase a computer, you&amp;#8217;d imagine that you&amp;#8217;re starting with a clean slate. However, this couldn&amp;#8217;t be further from the truth. Most computers are loaded with unwanted games and software (referred to as bloatware, since it bulks up what should be a clean slate). The most efficient way to truly free up your computer is to install a brand-new, store-bought copy of Windows onto your hard drive.&nbsp; This will wipe out everything that was on the hard drive (so make sure you save everything you wanted to keep on an external drive or disc) and leave a clean and smoothly operating system, free of unwanted, useless bloatware.&lt;/p&gt; &lt;p&gt;If you don’t feel like doing this, or don’t want to buy a new copy of Windows, you can try manually uninstalling the offending software, by going to Start &amp;gt; Control Panel &amp;gt; Programs and Features. From here you can go to each program you don’t want and click uninstall. For programs that are more deeply rooted, like anti-virus software, you can go to the developers website and search for the “complete uninstall” procedure and follow their steps to completely eradicate it from your system.&lt;/p&gt; &lt;h2&gt;Anti-virus and Security Software&lt;/h2&gt; &lt;p&gt;While many computers come with standard anti-virus software, it&amp;#8217;s important to make sure you have the right program to deal with potential intruders. That being said, you may want to consider forking out a few dollars for an all-encompassing anti-virus program, such as Norton Antivirus software, or save some money and get basic protection, with something like &lt;a href="http://free.avg.com/us-en/homepage"&gt;AVG Anti-Virus&lt;/a&gt; software.&lt;/p&gt; &lt;p&gt;Going together with anti-virus software is security scanning software. What you&amp;#8217;re looking for are programs such as malware that will make your computer run at a turtle&amp;#8217;s pace. By running a security scan, you can remove all the unwanted unintentionally downloaded programs off your hard drive before you get into computing that would release sensitive information.&lt;/p&gt; &lt;p&gt;Schedule a security scan for about once a week (you can set it to automatically begin when you want, in whatever increment you choose) to keep your computer free from malicious programs. Keep this software running in the background, and it will detect and quarantine any questionable and infected files that you may have just unintentionally downloaded and it will also warn you and deter you away from possibly unsafe sites.&lt;/p&gt; &lt;h2&gt;Back Up Software and Recovery&lt;/h2&gt; &lt;p&gt;Any computer with Windows will have system recovery loaded to it. Recovery restores your computer to a pre-existing state if the worst should happen.&lt;/p&gt; &lt;p&gt;If you drop your computer or it suddenly fails due to a power surge or something else, recovery will make your computer work again if possible. Backing up, on the other hand, is basically insurance for your computer. You can take all the files that you never want to lose, and you can put them on an external hard drive. Or if you don’t want to buy one of those, you can copy the files on to a DVD or CD and keep them in a safe place. These measures ensure that you will never have to fret over lost work.&lt;/p&gt; &lt;p&gt;Physical back-ups aren’t your only option – there are plenty of ways to &lt;a href="http://www.techradar.com/us/news/world-of-tech/roundup/how-to-choose-a-cloud-back-up-service-1087006"&gt;back up your info using an online cloud service&lt;/a&gt;, doing this will automatically back up your files as you make them, and no matter what happens to your computer, or backup drives, your files will be downloadable from their hard drives on their servers.&lt;/p&gt; &lt;h2&gt;Power Saving&lt;/h2&gt; &lt;p&gt;Make sure that, after registering Windows, you go into your personalization and check your power saving scheme. Here, you can &lt;a href="http://www.thewindowsclub.com/configure-power-plan-windows-8"&gt;configure your Windows 8 power plan settings and options&lt;/a&gt;. Choose how long until your computer turns off after it&amp;#8217;s idle, as well as setting a screen saver. Both of these things will save you on your energy bill and keep your computer running longer.&lt;/p&gt; &lt;p&gt;No matter what laptop you choose to buy, following a few simple steps for your new laptop will go a long way. Although you shouldn&amp;#8217;t expect problems on a brand new machine, it&amp;#8217;s all about peace of mind. You can use the computer at your leisure and not be worried about the problems that can compound over time.&lt;/p&gt;

    &lt;p&gt;As SSDs become increasingly affordable, making the switch is increasingly tempting. However, there are very few drives with over 512GB of capacity, and those that exist are still far from affordable. One solution that works well for desktops and workstation replacement laptops is putting an SSD in one drive bay and a mechanical HDD in another. This is a bit trickier with smaller laptops though. A couple years ago, I experimented with moving the primary HDD to the optical bay, and installing an SSD to get the best of both worlds: fast performance and extra storage for bulky but less-used files.&lt;/p&gt; &lt;p&gt;&amp;#8220;Ultrabook&amp;#8221; didn&amp;#8217;t even enter our vernacular until mid-2011, and at the time most laptops still came with a DVD drive. What&amp;#8217;s a modern laptop user to do if they want SSD performance and HDD storage?&lt;/p&gt; &lt;p&gt;&lt;a href="http://www.amazon.com/gp/product/B0062CHMZG/ref=as_li_ss_il?ie=UTF8&amp;amp;camp=1789&amp;amp;creative=390957&amp;amp;creativeASIN=B0062CHMZG&amp;amp;linkCode=as2&amp;amp;tag=nathhink-20"&gt;&lt;img class="alignright" style="border: 0px" alt="" src="http://ws-na.amazon-adsystem.com/widgets/q?_encoding=UTF8&amp;amp;ASIN=B0062CHMZG&amp;amp;Format=_SL160_&amp;amp;ID=AsinImage&amp;amp;MarketPlace=US&amp;amp;ServiceVersion=20070822&amp;amp;WS=1&amp;amp;tag=nathhink-20" width="160" height="102" border="0" /&gt;&lt;/a&gt;&lt;img style="border: none !important;margin: 0px !important" alt="" src="http://ir-na.amazon-adsystem.com/e/ir?t=nathhink-20&amp;amp;l=as2&amp;amp;o=1&amp;amp;a=B0062CHMZG" width="1" height="1" border="0" /&gt;&lt;/p&gt; &lt;p&gt;Enter the mSATA SSD: announced in 2009, mSATA SSDs started making their way into ultrabooks as manufacturers sought smaller sized components. In recent years some manufacturers have begun putting mSATA support into mid-sized laptops that still have a normal 2.5&amp;#8243; hard drive bay, too. Lenovo in particular has been leading this trend, with most recent ThinkPad and IdeaPad laptops supporting an mSATA drive in addition to the primary hard drive.&lt;/p&gt; &lt;p&gt;&lt;span id="more-5347"&gt;&lt;/span&gt;&lt;/p&gt; &lt;p&gt;The first thing to know about mSATA SSDs: they&amp;#8217;re small. Kingston donated &lt;a href="http://www.amazon.com/gp/product/B0062CHMZG/ref=as_li_ss_tl?ie=UTF8&amp;amp;camp=1789&amp;amp;creative=390957&amp;amp;creativeASIN=B0062CHMZG&amp;amp;linkCode=as2&amp;amp;tag=nathhink-20"&gt;a 64 GB mSATA SSD&lt;/a&gt; for this review, and when it arrived I was shocked by just how tiny the package is. It measures a mere 50 by 30 mm. The physical connector is identical to a mini-PCI-E slot, like those used by WiFi and WWAN cards, but does require that a SATA channel be connected to the slot, so just any mini-PCI-E slot won&amp;#8217;t do.&lt;/p&gt; &lt;h1&gt;Installation&lt;/h1&gt; &lt;p&gt;Installing an mSATA SSD takes a lot more time and care than installing a regular 2.5&amp;#8243; drive into most laptops. To install it into my ThinkPad X230 took about an hour, vs. a couple minutes to swap out the 2.5&amp;#8243; drive. Each computer will require different, very specific steps to install. Look up your computer&amp;#8217;s service manual (typically available from the manufacturer&amp;#8217;s support site) and find the instructions for installing an mSATA drive.&lt;/p&gt; &lt;div style="width: 610px" class="wp-caption aligncenter"&gt;&lt;img alt="" src="https://i.minus.com/irSxLkoS0GDZ4.gif" width="600" height="449" /&gt;&lt;p class="wp-caption-text"&gt;The mSATA SSD installation flipbook. I&amp;#8217;ve found that it&amp;#8217;s almost always easier to just do a fresh install of Windows on a new SSD. Trying to image from a large disk to a smaller disk is generally a pain to do, and Windows will also automatically configure certain things differently if you have an SSD, to improve its performance.&lt;/p&gt; &lt;p&gt;Working from a fresh install, you&amp;#8217;ll want to configure your user files (My Documents, My Pictures, etc.) to reside on the HDD. Instructions for this &lt;a href="http://blog.superuser.com/2011/06/30/installing-the-optical-bay-hdd/#config"&gt;are exactly the same as with my previous SSD+HDD Caddy instructions&lt;/a&gt;.&lt;/p&gt; &lt;h1&gt;Performance&lt;/h1&gt; &lt;p&gt;In most cases, an mSATA SSD will perform just as well in a given computer as a regular 2.5&amp;#8243; SATA drive, if they have the same basic specs. For comparison, I used a similar Kingston SSD in the 2.5&amp;#8243; form factor, the &lt;a href="http://www.amazon.com/gp/product/B00A1ZTZOG/ref=as_li_ss_tl?ie=UTF8&amp;amp;camp=1789&amp;amp;creative=390957&amp;amp;creativeASIN=B00A1ZTZOG&amp;amp;linkCode=as2&amp;amp;tag=nathhink-20"&gt;SSDnow V300&lt;/a&gt;, which does take advantage of the SATA III connector in the full HDD slot. Note that these comparisons are just to give an idea of the relative performance in this case &amp;#8211; with an mSATA SSD and slot that both support SATA III, you could easily see better performance from an mSATA SSD than some 2.5&amp;#8243; SSDs.&lt;/p&gt; &lt;h2&gt;AS SSD Benchmark&lt;/h2&gt; &lt;div style="width: 650px" class="wp-caption aligncenter"&gt;&lt;a href="http://i.stack.imgur.com/EQfe2.png"&gt;&lt;img alt="" src="http://i.stack.imgur.com/EQfe2l.png" width="640" height="325" /&gt;&lt;/a&gt;&lt;p class="wp-caption-text"&gt;mSATA SSD on left, 2.5&amp;#8243; SSD on right&lt;/p&gt;&lt;/div&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;h2&gt;Crystal Disk Mark Benchmark&lt;/h2&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;div style="width: 650px" class="wp-caption aligncenter"&gt;&lt;a href="http://i.stack.imgur.com/DEbAH.png"&gt;&lt;img class=" " alt="" src="http://i.stack.imgur.com/DEbAHl.png" width="640" height="293" /&gt;&lt;/a&gt;&lt;p class="wp-caption-text"&gt;CrystalDiskMark benchmarks. mSATA SSD on left, 2.5&amp;#8243; SSD on right&lt;/p&gt;&lt;/div&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;h1&gt;Making sure you have what you need&lt;/h1&gt; &lt;p&gt;If you&amp;#8217;re interested in pursuing an mSATA SSD solution, the first thing to do is make sure your laptop supports it! Most Lenovo laptops (both ThinkPads and IdeaPads) since about 2010 have mSATA slots. Some other manufacturers have them too, although less consistently. If you already have the laptop, try finding the specific service manual for your computer, and look for information about the mSATA drive, or search for &amp;#8220;&amp;lt;laptop name&amp;gt; mSATA&amp;#8221; online. If you haven&amp;#8217;t purchased a laptop yet, try to find this information before buying!&lt;/p&gt; &lt;p&gt;Once you have the laptop taken care of, you&amp;#8217;ll need to select an SSD.     &lt;p&gt;&lt;span style="font-size: 13px"&gt;I recently moved to a bigger city, and of course the first thing that I setup was my internet connection. &nbsp;I&amp;#8217;ve had the same ISP for a couple of years now, and they&amp;#8217;ve been alright. &nbsp;I&amp;#8217;ve noticed drops in service here and there but overall it was a tolerable&nbsp;experience. &nbsp;However, as soon as I started watching my online shows in the new place, I noticed some dramatic changes in my viewing experience. &nbsp;There were long periods of waiting for the shows to buffer, and I felt like I was relapsing to my younger days of &lt;/span&gt;&lt;a style="font-size: 13px" href="http://www.youtube.com/watch?v=FG1AQcGGSec"&gt;dial-up&lt;/a&gt;&lt;span style="font-size: 13px"&gt;. &nbsp;I got so frustrated one night I even tweeted about it:&lt;/span&gt;&lt;/p&gt; &lt;blockquote class="twitter-tweet"&gt;Just took me two hours to watch a 1 hour movie. Why I&amp;#8217;m leaving @&lt;a href="https://twitter.com/coxcomm"&gt;coxcomm&lt;/a&gt;. — Krono$ (@SuperKronos) &lt;a href="https://twitter.com/SuperKronos/status/295770985620176896"&gt;January 28, 2013&lt;/a&gt;&lt;/blockquote&gt; &lt;p&gt;I finally decided to&nbsp;switch&nbsp;over and give DSL a shot, but I didn&amp;#8217;t cancel my Cable. &nbsp;I figured I&amp;#8217;d do a little testing and really see who was the better service after all. &nbsp;I put together a pretty extensive document of my results. &nbsp;You can find that document and all the data I captured and used in a link below. This blog post is a summary of what I found to be the important parts.&lt;/p&gt; &lt;p&gt;&lt;span id="more-5319"&gt;&lt;/span&gt;&lt;/p&gt; &lt;p&gt;Before I continue there are a few things that I need to let you know. &nbsp;In all fairness to Cox (the cable provider), they did reach out to me via twitter to the frustration I vented above. &nbsp;They said that using a newer version of a cable modem (DOCSYS 3.0) would lead to faster speeds and better bandwidth, but I didn&amp;#8217;t really want to spend the $60-$90 just for a study when I&amp;#8217;m already committed to DSL. &nbsp;Perhaps if they&amp;#8217;re willing to donate a new modem and a few weeks of free cable, I&amp;#8217;ll redo the testing. &nbsp;CenturyLink (the DSL provider) become my new primary source of internet. &nbsp;I know that when testing like this, the test network shouldn&amp;#8217;t be used during. &nbsp;However, my life is on the internet, and I wasn&amp;#8217;t about to give it up just to avoid the randomized testing. &nbsp;So these results include daily usage on the DSL connection. &nbsp;The Cox cable line was untouched during testing with the exception of any Windows (or other installed apps) automatic updates. &nbsp;Finally, the results of this test&nbsp;&lt;em&gt;may not be the same as what you experience&lt;/em&gt;. &nbsp;This is a quantitative and qualitative approach to testing in my geographical area.&nbsp; The Cable was a 25 Mbit/s advertised connection and DSL was a 12 Mbit/s advertised connection.&lt;/p&gt; &lt;p&gt;So here are a few interesting results:&lt;/p&gt; &lt;h4&gt;During Peak Hours DSL outperforms Cable in Download Speeds&lt;/h4&gt; &lt;p&gt;I considered peak hours to be between 17:00 and 23:00 local time or basically the hours that I&amp;#8217;m most active online during the week. &nbsp;After testing I noticed that the download speeds were on average and consistently far better for DSL than that of Cable. &nbsp;Here&amp;#8217;s the average speed over roughly two weeks. &nbsp;There&amp;#8217;s a whole 2 Mbit/s speed difference between the two.&lt;/p&gt; &lt;p&gt;&lt;img title="" alt="enter image description here" src="http://i.stack.imgur.com/0A5Ia.png" /&gt;&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;I put together a box plot that shows where roughly %50 of the values are for each. &nbsp;The DSL connection was more concentrated in one central area, and was faster than the Cable!&lt;/p&gt; &lt;p&gt;&lt;a href="http://i.stack.imgur.com/ioma7.jpg"&gt;&lt;img class="alignnone" alt="" src="http://i.stack.imgur.com/ioma7.jpg" width="572" height="513" /&gt;&lt;/a&gt;&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;Finally I decided to put together a confidence interval where I could be %95 sure where the range of speed would be during those peak hours. &nbsp;This is what I found:&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;&lt;img title="" alt="enter image description here" src="http://i.stack.imgur.com/rVFCp.png" /&gt;&lt;/p&gt; &lt;p&gt;Not only was the interval smaller (less variation of the speeds during the hours), but it was overall faster than Cable! &nbsp;Before testing I assumed that Cable and DSL were going to be pretty close especially since the DSL connection was half the advertised speed as the Cable, but for the fact that DSL excelled beyond Cable blew my mind.&lt;/p&gt; &lt;h4&gt;&nbsp;DSL Upload speeds aren&amp;#8217;t speedy at all&lt;/h4&gt; &lt;p&gt;One of the things that Cable did do well was in Upload speeds. &nbsp;Of course that&amp;#8217;s because&nbsp;DSL only&nbsp;offered a 800 kBit/s up speed, but here&amp;#8217;s a few graphs to show just&nbsp;how bad the comparison was:&lt;/p&gt; &lt;p&gt;&lt;img title="" alt="enter image description here" src="http://i.stack.imgur.com/lXsVL.png" /&gt;&lt;/p&gt; &lt;p&gt;&amp;nbsp;&lt;/p&gt; &lt;p&gt;This will lead to some bandwidth issues, especially if you&amp;#8217;re gaming. &nbsp;Dropbox for iOS decided to start uploading my pictures while I was playing a game. &nbsp;This led to some pretty horrible ping times and me pulling out&nbsp;my hair wondering what was going on for about a 1/2 hour. &nbsp;The take away from this is, if you&amp;#8217;re&nbsp;uploading&nbsp;a lot of data on a DSL connection, don&amp;#8217;t play games while your at it.&lt;/p&gt; &lt;h4&gt;Reliability and speed set aside: I still didn&amp;#8217;t get 100% of my advertised speed&lt;/h4&gt; &lt;p&gt;Neither Cox nor Century Link provided the full 100%&lt;/p&gt; &lt;p&gt;&lt;img title="" alt="enter image description here" src="http://i.stack.imgur.com/3R5Nq.png" /&gt;&lt;/p&gt; &lt;p&gt;Cable&amp;#8217;s 27% performance during peak hours is dismal and really unacceptable, especially since Cox did so much better according to the &lt;a href="http://www.fcc.gov/measuring-broadband-america/2013/February"&gt;FCC&amp;#8217;s national testing&lt;/a&gt;. &nbsp;DSL you&amp;#8217;re not off the hook either. &nbsp;71% is bad as well, just not as bad as Cable.&lt;/p&gt; &lt;h4&gt;At the end of the day, how did I feel about the service?&lt;/h4&gt; &lt;p&gt;Numbers are great, and&nbsp;they qualify what I as a customer feel, but in the end they really don&amp;#8217;t matter. &nbsp;What matters most is whether or not I get frustrated with my current service. &nbsp;I have to say, that while DSL hasn&amp;#8217;t given me the speeds that I expected and hoped for, the reliability has been a big plus. &nbsp;I&amp;#8217;ve only noticed lags here and there, and I haven&amp;#8217;t (yet) had to sit and wait for buffering that took just as long as the actual show that I was watching.&lt;/p&gt; &lt;p&gt;Want to look at the full results? &nbsp;Here&amp;#8217;s &lt;a href="https://www.dropbox.com/s/hlf7w2k53wktuwm/Cable%20and%20DSL.pdf"&gt;a link to the .pdf document&lt;/a&gt; I put together for the testing I did. &nbsp;Here&amp;#8217;s &lt;a href="https://www.dropbox.com/s/juvnma82by55enj/Cox%20vs%20Century%20Link.zip"&gt;another link to all of the data&lt;/a&gt; if you want to mess around with it yourself.&lt;/p&gt;

You can use flat or dotted-quad (or dotted-triple, dotted-double, or even dotted-single) format, and for each one, you can use (or even mix and match) decimal, octal, and hexadecimal. For example, you can ping &lt;code&gt;google.com&lt;/code&gt; in the following ways:&lt;/p&gt; &lt;ul&gt; &lt;li&gt;&lt;code&gt;google.com&lt;/code&gt;&amp;#160; (domain name) &lt;/li&gt; &lt;li&gt;&lt;code&gt;74.125.226.4&lt;/code&gt;&amp;#160; (dotted decimal) &lt;/li&gt; &lt;li&gt;&lt;code&gt;1249763844&lt;/code&gt;&amp;#160; (flat decimal) &lt;/li&gt; &lt;li&gt;&lt;code&gt;0112.0175.0342.0004&lt;/code&gt;&amp;#160; (dotted octal) &lt;/li&gt; &lt;li&gt;&lt;code&gt;011237361004&lt;/code&gt;&amp;#160; (flat octal) &lt;/li&gt; &lt;li&gt;&lt;code&gt;0x4A.0x7D.0xE2.0x04&lt;/code&gt;&amp;#160; (dotted hex) &lt;/li&gt; &lt;li&gt;&lt;code&gt;0x4A7DE204&lt;/code&gt;&amp;#160; (flat hex) &lt;/li&gt; &lt;li&gt;&lt;code&gt;74.0175.0xe2.4&lt;/code&gt;&amp;#160; (ಠ_ಠ)&lt;/li&gt; &lt;/ul&gt; &lt;/blockquote&gt; &lt;p&gt;&amp;#160;&lt;/p&gt; &lt;p&gt;So now you can play around with your friends with your ping-fu mad skilz:&lt;/p&gt; &lt;p&gt;&lt;img alt="enter image description here" src="http://i.stack.imgur.com/UFMti.png" /&gt;&lt;/p&gt;
