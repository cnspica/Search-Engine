            <div class="post-text" itemprop="text"> 
             <p>The Coursera cryptography 1 course says: </p> 
             <blockquote> 
              <p>If you want to use MAC-then-encrypt mode then you should either use randomized-CTR or randomized-CBC. And if you use randomized-CTR then one-time-MAC is sufficient.</p> 
             </blockquote> 
             <p>Here is the slide: <a href="http://i.stack.imgur.com/5G6OA.png" rel="nofollow"><img src="http://i.stack.imgur.com/5G6OA.png" alt="MAC-then-encrypt"></a> I have two question regarding this:</p> 
             <ol> 
              <li>How we can use one-time-MAC without negotiating the different keys for each message?</li> 
              <li>What is wrong with randomized-CBC that doesn't allow us to use with one-time-MAC?</li> 
             </ol>
