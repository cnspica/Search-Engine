            <div class="post-text" itemprop="text"> 
             <p><em>"My manager asked me to finish the project as early as possible"</em> - Here there is no fixed deadline available but the manager wants me to finish the project soon. Consider the case where there is a deadline but I can take <em>as much time as I want</em> before the deadline to finish the project. In this case, is it appropriate for a person to say, "<em>You can finish the project <strong>as late as possible</strong> (meaning, you have time to do), but do it before the deadline expires</em>". </p> 
             <p>I am looking for an exact opposite phrase for <em>as early as possible</em>. If it is not there, is there a phrase which can capture what I have mentioned in the above mentioned paragraph?</p>

           <td class="answercell"> 
            <div class="post-text" itemprop="text"> 
             <p>In business context, I would say, "Finish the project when (whenever) it is <a href="http://dictionary.cambridge.org/dictionary/english/convenient">convenient</a> for you before the deadline". </p> 
             <p>I have never heard someone say, "Please finish it as late as possible" which would be understood as the speaker wants you to finish the project at the latest moment before the deadline. </p> 
             <p>But, if you say "when it is convenient for you", it doesn't impose any urgency and the speaker can wait until it is finished before the deadline. </p>

           <td class="answercell"> 
            <div class="post-text" itemprop="text"> 
             <p><em>The question is a little unclear about the request. Some readers have interpreted it as "finish at any time you like", some as "finish any time you like, as long as it is before the deadline" and some as "start as late as you can, while still finishing before the deadline." I have read it as the latter meaning. This is not an unusual approach in projects - e.g. if there is a risk that the requirements are still changing; there is a risk the project might be cancelled in the short term; if you are doing some operation which is more efficient in bulk so it is better to wait for as many inputs as possible to arrive; or if it has to be done as the last task - like sweeping the sawdust off the floor.</em></p> 
             <p><a href="http://idioms.thefreedictionary.com/at+the+last+minute" rel="nofollow">At the last minute</a> is an idiom for not starting until you absolutely have to. It tends to have negative connotations - i.e. that you were procrastinating, rather than deliberately timing it.</p> 
             <p>In the manufacturing industry, <a href="https://en.wikipedia.org/wiki/Just-in-time_manufacturing" rel="nofollow">Just In Time</a> refers to a philosophy of having all the materials arrive to be worked on as late as possible while still being in time. It is a deliberate effort to reduce the costs of maintaining a big inventory. So you could say "do this just in time to meet the deadline."</p>

             <blockquote> 
              <p>Please have this project done by mm/dd/yyyy</p>

             <p>or</p> 
             <blockquote> 
              <p>This project should be finished no later than mm/dd/yyyy</p> 
             </blockquote> 
             <p>or</p> 
             <blockquote> 
              <p>This project cannot go beyond mm/dd/yyyy</p> 
             </blockquote> 
             <p>or (credit to <a href="http://english.stackexchange.com/users/107111">Paparazzi</a>)</p> 
             <blockquote> 
              <p>The task has a deadline of mm/dd/yyyy</p>

             <p>I think the question a lot of people are grappling with is "why". Most often, this situation arises when the person has higher priority tasks, e.g.</p> 
             <blockquote> 
              <p>Task B needs doing urgently, whereas you have all the way up to the 32nd of Febtember to complete task A</p> 
             </blockquote> 
             <p>If the task requires other resources that are better spent elsewhere immediately, the boss could say </p> 
             <blockquote> 
              <p>Don't worry about task A for now. It'll wait.</p> 
             </blockquote> 
             <p>Lastly, in giving out the task, the boss might describe it as a "[time] filler", to be done in those moments where it's not possible to do anything else, for example:</p> 
             <blockquote> 
              <p>I'd like you to do task A - it's only a filler, doesn't need to be submitted until 32nd of Febtember.</p>

           <td class="votecell"> 
            <div class="vote"> 
             <input type="hidden" name="_id_" value="325353"> 
             <a class="vote-up-off" title="This answer is useful">up vote</a> 
             <span itemprop="upvoteCount" class="vote-count-post ">1</span>

           <td class="answercell"> 
            <div class="post-text" itemprop="text"> 
             <p>The opposite of "as early as possible" is either "as late as possible" or "as early as is necessary" (they both mean the same thing).</p> 
             <p>It is used, especially in project management.</p> 
             <p>It means "in enough time to finish by the deadline, but no earlier".</p> 
             <p>There are lots of reasons to schedule that way:</p> 
             <ul> 
              <li><p>One activity might interrupt another activity - so you don't start the new activity until you have to. </p></li> 
              <li><p>You may want to minimise the time between finishing one activity and starting another activity - so you don't start the first activity until you have to.</p></li> 
              <li><p>You may be hoping that an activity can be cancelled, or worried that requirements may change - so you don't start the activity until you have to.</p></li> 
             </ul> 
             <p>In between "as early as possible" and "as late as possible" is "by &lt;date&gt;". As other people have said, that means, "it doesn't matter when you do it, so long as its done by &lt;date&gt;."</p> 
             <p>But if you say "take as much time as you want", that's quite different. It means "Take enough time to do a good job. This job is too important to rush."</p>

           <td class="answercell"> 
            <div class="post-text" itemprop="text"> 
             <p>Not <em>quite</em> what you're asking, but you did say "opposite". In developing situations, you might be very interested in the latest available information; for example, a report about a military or political situation, or a sports match. In this case, you might, in order of decreasing formality, be instructed <em>to be</em>, or <em>to include information which is</em>:</p> 
             <ul> 
              <li><strong>as current as possible</strong></li> 
              <li><strong>as up-to-date as possible</strong></li> 
              <li><strong>as hot as you can [get]</strong></li> 
             </ul> 
             <p>Answering your question only strictly, then there is no true opposite, since the phrase you offered, "as early as possible" (or more typically, "as soon as possible", ASAP) demands urgency. Without urgency, there's nothing to specify. So the opposite <em>sense</em> is simply to add "it's not urgent", or "it's not a priority".</p>

Also, to say "<em>do a job in latest time possible but before a deadline</em>" is not something any boss would say to an employee. Its a contradictory statement, almost an oxymoron.</p> 
             <p>The work has to be done on time but there is no pressure for it to be done soon. May be this is what you are looking for :- </p> 
             <p><strong><em>take your own sweet time</em></strong></p> 
             <p><em>"Take your own sweet time but it should not be left undone."</em> </p>
