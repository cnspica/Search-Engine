        <p><b>S</b>calable <b>V</b>ector <b>G</b>raphics commonly known as SVG is a XML based format to draw vector images. It is used to draw two dimentional vector images.</p> 
        <p>This tutorial will teach you basics of SVG. Tutorial contains chapters discussing all the basic components of SVG with suitable examples.</p> 
        <h1>Audience</h1> 
        <p>This reference has been prepared for the beginners to help them to understand the basic concepts related to SVG. This tutorial will give you enough understanding on SVG from where you can take yourself to a higher level of expertise.</p> 
        <h1>Prerequisites</h1> 
        <p>Before proceeding with this tutorial you should have basic knowledge of XML, HTML and Javascript.</p>
