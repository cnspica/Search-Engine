        <h1>Google Maps - Getting Started</h1> 
        <h2>What are Google Maps?</h2> 
        <p>Google Maps is a free web mapping service by Google that provides various types of geographical information. Using Google Maps, one can.</p> 
        <ul class="list"> 
         <li><p>Search for places or get directions from one place to another.</p></li> 
         <li><p>View and navigate through horizontal and vertical panoramic street level images of various cities around the world.</p></li> 
         <li><p>Get specific information like traffic at a particular point.</p></li> 
        </ul> 
        <p>Google Maps provides an API using which you can customize the maps and the information displayed on them. This chapter explains how to load a simple Google Map on your web page using HTML and JavaScript.</p> 
        <h2>Steps to Load the Map on a Webpage</h2> 
        <p>Follow the steps given below to load a map on your webpage −</p> 
        <h3>Step 1 : Create an HTML Page</h3> 
        <p>Create a basic HTML page with head and body tags as shown below −</p> 
        <pre class="prettyprint notranslate">
&lt;!DOCTYPE html&gt;
&lt;html&gt;
   
   &lt;head&gt;
   &lt;/head&gt;
   
   &lt;body&gt;
   ..............
   &lt;/body&gt;
   
&lt;/html&gt;
</pre> 
        <h3>Step 2 : Load the API</h3> 
        <p>Load or include the Google Maps API using the script tag as shown below −</p> 
        <pre class="prettyprint notranslate ">
&lt;!DOCTYPE html&gt;
&lt;html&gt;
   
   &lt;head&gt;
      &lt;script src = "http://maps.googleapis.com/maps/api/js"&gt;&lt;/script&gt;
   &lt;/head&gt;
   
   &lt;body&gt;
   ..............
   &lt;/body&gt;
   
&lt;/html&gt;
</pre> 
        <h3>Step 3 : Create the Container</h3> 
        <p>To hold the map, we have to create a container element, generally the &lt;div&gt; tag (a generic container) is used for this purpose. Create a container element and define its dimensions as shown below −</p> 
        <pre class="prettyprint notranslate">
&lt;div id = "sample" style = "width:900px; height:580px;"&gt;&lt;/div&gt;
</pre> 
        <h3>Step 4 : Map Options</h3> 
        <p>Before initializing the map, we have to create a <b>mapOptions</b> object (it is created just like a literal) and set values for map initialization variables. A map has three main options, namely, <b>centre</b>, <b>zoom</b>, and <b>maptypeid</b>.</p> 
        <ul class="list"> 
         <li><p><b>centre</b> − Under this property, we have to specify the location where we want to centre the map. To pass the location, we have to create a <b>LatLng</b> object by passing the latitude and longitudes of the required location to the constructor.</p></li> 
         <li><p><b>zoom</b> − Under this property, we have to specify the zoom level of the map.</p></li> 
         <li><p><b>maptypeid</b>− Under this property, we have to specify the type of the map we want. Following are the types of maps provided by Google −</p> 
          <ul class="list"> 
           <li>ROADMAP (normal, default 2D map)</li> 
           <li>SATELLITE (photographic map)</li> 
           <li>HYBRID (combination of two or more other types)</li> 
           <li>TERRAIN (map with mountains, rivers, etc.)</li> 
          </ul> </li> 
        </ul> 
        <p>Within a function, say, <b>loadMap()</b>, create the mapOptions object and set the required values for center, zoom, and mapTypeId as shown below.</p> 
        <pre class="prettyprint notranslate">
function loadMap() {
   var mapOptions = {
      center:new google.maps.LatLng(17.240498, 82.287598), 
      zoom:9, 
      mapTypeId:google.maps.MapTypeId.ROADMAP
   };
}
</pre> 
        <h3>Step 5 : Create a Map Object</h3> 
        <p>You can create a map by instantiating the JavaScript class called <b>Map</b>. While instantiating this class, you have to pass an HTML container to hold the map and the map options for the required map. Create a map object as shown below.</p> 
        <pre class="result notranslat">
var map = new google.maps.Map(document.getElementById("sample"),mapOptions);
</pre> 
        <h3>Step 6 : load the map</h3> 
        <p>Finally load the map by calling the loadMap() method or by adding DOM listener.</p> 
        <pre class="prettyprint notranslate">
google.maps.event.addDomListener(window, 'load', loadMap);
                    or
&lt;body onload = "loadMap()"&gt;
</pre> 
        <h2>Example</h2> 
        <p>The following example shows how to load the roadmap of the city named Vishakhapatnam with a zoom value of 12.</p> 
        <pre class="prettyprint notranslate tryit">
&lt;!DOCTYPE html&gt;
&lt;html&gt;
   
   &lt;head&gt;
      &lt;script src = "http://maps.googleapis.com/maps/api/js"&gt;&lt;/script&gt;
      
      &lt;script&gt;
         function loadMap() {
			
            var mapOptions = {
               center:new google.maps.LatLng(17.609993, 83.221436), 
               zoom:12, 
               mapTypeId:google.maps.MapTypeId.ROADMAP
            };
				
            var map = new google.maps.Map(document.getElementById("sample"),mapOptions);
         }
			
         google.maps.event.addDomListener(window, 'load', loadMap);
      &lt;/script&gt;
      
   &lt;/head&gt;
   
   &lt;body&gt;
      &lt;div id = "sample" style = "width:580px; height:400px;"&gt;&lt;/div&gt;
   &lt;/body&gt;
   
&lt;/html&gt;
</pre> 
        <p>It produces the following output −</p> 
        <iframe class="result" src="/google_maps/src/sample.htm" width="600px" height="420px"></iframe> 
        <h1>Google Maps - Types</h1> 
        <p>In the previous chapter, we discussed how to load a basic map. Here, we will see how to select a required map type.</p> 
        <h2>Types of Maps</h2> 
        <p>Google Maps provides four types of maps. They are −</p> 
        <ul class="list"> 
         <li><p><b>ROADMAP</b> − This is the default type. If you haven't chosen any of the types, this will be displayed. It shows the street view of the selected region.</p></li> 
         <li><p><b>SATELLITE</b> − This is the map type that shows the satellite images of the selected region.</p></li> 
         <li><p><b>HYBRID</b> − This map type shows the major streets on satellite images.</p></li> 
         <li><p><b>TERRAIN</b> − This is the map type that shows the terrain and vegetation</p></li> 
        </ul> 
        <h3>Syntax</h3> 
        <p>To select one of these map types, you have to mention the respective map type id in the map options as shown below −</p> 
        <pre class="prettyprint notranslate">
var mapOptions = {
   mapTypeId:google.maps.MapTypeId.<b>Id of the required map type</b>
};
</pre>

        <h1>Google Maps - Zoom</h1> 
        <h2>Increase/Decrease the Zoom Value</h2> 
        <p>You can increase or decrease the zoom value of a map by modifying the value of the <b>zoom</b> attribute in the the map options.</p> 
        <h3>Syntax</h3> 
        <p>We can increase or decrease the zoom value of the map using the zoom option. Given below is the syntax to change the zoom value of the current map.</p> 
        <pre class="prettyprint notranslate">
var mapOptions = {
   <b>zoom:required zoom value</b>
};
</pre>

        <h1>Google Maps - Localization</h1> 
        <p>By default, the city names and option names given on the map will be in English. If required, we can display such information in other languages as well. This process is known as <b>localization</b>. In this chapter, we will learn how to localize a map.</p> 
        <h2>Localizing a Map</h2> 
        <p>You can customize (localize) the language of the map by specifying the language option in the URL as shown below.</p> 
        <pre class="prettyprint notranslate">
&lt;script src = "https://maps.googleapis.com/maps/api/js?language=zh-Hans"&gt;&lt;/script&gt;
</pre> 
        <h3>Example</h3> 
        <p>Here is an example that shows how to localize GoogleMaps. Here you can see a roadmap of China that is localized to Chinese language.</p> 
        <pre class="prettyprint notranslate tryit">
&lt;!DOCTYPE html&gt;
&lt;html&gt;
   &lt;head&gt;
      &lt;script src = "https://maps.googleapis.com/maps/api/js?language=zh-Hans"&gt;&lt;/script&gt;
      
      &lt;script&gt;
         function loadMap() {
			
            var mapOptions = {
               center:new google.maps.LatLng(32.870360, 101.645508),
               zoom:9, 
               mapTypeId:google.maps.MapTypeId.ROADMAP
            };
				
            var map = new google.maps.Map(document.getElementById("sample"),mapOptions);
         }
      &lt;/script&gt;

        <h1>Google Maps - Info Window</h1> 
        <p>In addition to markers, polygons, polylines, and other geometrical shapes, we can also draw an Info Window on the map. This chapter explains how to use the Info Window.</p> 
        <h2>Adding a Window</h2> 
        <p>Info Window is used to add any kind of information to the map. For instance, if you want to provide information about a location on the map, you can use an info window. Usually the info window is attached to a marker. You can attach an info window by instantiating the <b>google.maps.InfoWindow</b> class. It has the following properties −</p> 
        <ul class="list"> 
         <li><p><b>Content</b> − You can pass your content in String format using this option.</p></li> 
         <li><p><b>position</b> − You can choose the position of the info window using this option.</p></li> 
         <li><p><b>maxWidth</b> − By default, the info window's width will be stretched till the text is wrapped. By specifying maxWidth, we can restrict the size of the info window horizontally.</p></li> 
        </ul> 
        <h3>Example</h3> 
        <p>The following example shows how to set the marker and specify an info window above it −</p> 
        <pre class="prettyprint notranslate tryit">
&lt;!DOCTYPE html&gt;
&lt;html&gt;
   
   &lt;head&gt;
      &lt;script src = "http://maps.googleapis.com/maps/api/js"&gt;&lt;/script&gt;
      
      &lt;script&gt;
         function loadMap() {
			
            var mapOptions = {
               center:new google.maps.LatLng(17.433053, 78.412172),
               zoom:5
            }
            
            var map = new google.maps.Map(document.getElementById("sample"),mapOptions);
            
            var marker = new google.maps.Marker({
               position: new google.maps.LatLng(17.433053, 78.412172),
               map: map,
               draggable:true,
               icon:'/scripts/img/logo-footer.png'
            });
            
            marker.setMap(map);
            
            var infowindow = new google.maps.InfoWindow({
               content:"388-A , Road no 22, Jubilee Hills, Hyderabad Telangana, INDIA-500033"
            });
				
            infowindow.open(map,marker);
         }
      &lt;/script&gt;
      
   &lt;/head&gt;
   
   &lt;body onload = "loadMap()"&gt;
      &lt;div id = "sample" style = "width:580px; height:400px;"&gt;&lt;/div&gt;
   &lt;/body&gt;
   
&lt;/html&gt;
</pre>

        <h1>Google Maps - Symbols</h1> 
        <p>In addition to markers, polygons, polylines, and other geometrical shapes, we can also add predefined vector images(symbols) on a map. This chapter explains how to use the symbols provided by Google Maps.</p> 
        <h2>Adding a Symbol</h2> 
        <p>Google provides various vector-based images (symbols) that can be used on a marker or a polyline. Just like other overlays, to draw these predefined symbols on a map, we have to instantiate their respective classes. Given below is a list of predefined symbols provided by Google and their class names −</p> 
        <ul class="list"> 
         <li><p><b>Circle</b> − google.maps.SymbolPath.CIRCLE</p></li> 
         <li><p><b>Backward Pointing arrow (closed)</b> − google.maps.SymbolPath.BACKWARD_CLOSED_ARROW</p></li> 
         <li><p><b>Forward Pointing arrow (closed)</b> − google.maps.SymbolPath.FORWARD_CLOSED_ARROW</p></li> 
         <li><p><b>Forward Pointing arrow (open)</b> − google.maps.SymbolPath.CIRCLE</p></li> 
         <li><p><b>Backward Pointing arrow (open)</b> − google.maps.SymbolPath.CIRCLE</p></li> 
        </ul> 
        <p>These symbols have the following properties − path, fillColor, fillOpacity, scale, stokeColor, strokeOpacity, and strokeWeight.</p> 
        <h3>Example</h3> 
        <p>The following example demonstrates how to draw predefined symbols on Google Map.</p> 
        <pre class="prettyprint notranslate tryit">
&lt;!DOCTYPE html&gt;
&lt;html&gt;
   
   &lt;head&gt;
      &lt;script src = "http://maps.googleapis.com/maps/api/js"&gt;&lt;/script&gt;
      
      &lt;script&gt;
         function loadMap() {

               center:new google.maps.LatLng(17.433053, 78.412172),
               zoom:5
            }
            
            var map = new google.maps.Map(document.getElementById("sample"),mapOptions);
            
            var marker = new google.maps.Marker({
               position: map.getCenter(),
               
               icon: {
                  path: google.maps.SymbolPath.BACKWARD_CLOSED_ARROW,
                  scale: 5,
                  strokeWeight:2,
                  strokeColor:"#B40404"
               },
					
               draggable:true,
               map: map,
            });

   &lt;/head&gt;

        <h1>Google Maps - Events</h1> 
        <p>The Google Maps JavaScript program can respond to various events generated by the user. This chapter provides examples demonstrating how to perform event handling while working with Google Maps.</p> 
        <h2>Adding an Event Listener</h2> 
        <p>You can add an event listener using the method <b>addListener()</b>. It accepts parameters such as object name on which we want to add the listener, name of the event, and the handler event.</p> 
        <h3>Example</h3> 
        <p>The following example shows how to add an event listener to a marker object. The program raises the zoom value of the map by 5 each time we double-click on the marker.</p> 
        <pre class="prettyprint notranslate tryit">
&lt;!DOCTYPE html&gt;
&lt;html&gt;
   
   &lt;head&gt;
      &lt;script src = "http://maps.googleapis.com/maps/api/js"&gt;&lt;/script&gt;
      
      &lt;script&gt;
         var myCenter = new google.maps.LatLng(17.433053, 78.412172);
         function loadMap(){
			
            var mapProp = {
               center: myCenter,
               zoom:5,
               mapTypeId: google.maps.MapTypeId.ROADMAP
            };
            
            var map = new google.maps.Map(document.getElementById("googleMap"),mapProp);
            
            var marker = new google.maps.Marker({
               position: myCenter,
               title:'Click to zoom'
            });
            
            marker.setMap(map);
            
            //Zoom to 7 when clicked on marker
            google.maps.event.addListener(marker,'click',function() {
               map.setZoom(9);
               map.setCenter(marker.getPosition());
            });
         }
      &lt;/script&gt;
      
   &lt;/head&gt;
   
   &lt;body onload = "loadMap()"&gt;
      &lt;div id = "googleMap" style = "width:580px; height:400px;"&gt;&lt;/div&gt;
   &lt;/body&gt;
   
&lt;/html&gt;
</pre>

        <h2>Removing the Listener</h2> 
        <p>You can remove an existing listener using the method <b>removeListener()</b>. This method accepts the listener object, therefore we have to assign the listener to a variable and pass it to this method.</p> 
        <h3>Example</h3> 
        <p>The following code shows how to remove a listener −</p> 
        <pre class="prettyprint notranslate tryit">
&lt;!DOCTYPE html&gt;
&lt;html&gt;
   
   &lt;head&gt;
      &lt;script src = "http://maps.googleapis.com/maps/api/js"&gt;&lt;/script&gt;
      
      &lt;script&gt;
         var myCenter = new google.maps.LatLng(17.433053, 78.412172);
         function loadMap(){
			
            var mapProp = {
               center:myCenter,
               zoom:4,
               mapTypeId:google.maps.MapTypeId.ROADMAP
            };
            
            var map = new google.maps.Map(document.getElementById("googleMap"),mapProp);
            
            var marker = new google.maps.Marker({
               position:myCenter,
            });
            
            marker.setMap(map);
            
            var infowindow = new google.maps.InfoWindow({
               content:"Hi"
            });
            
            var myListener = google.maps.event.addListener(marker, 'click', function() {
               infowindow.open(map,marker);
            });
				
            google.maps.event.removeListener(myListener);
         }
      &lt;/script&gt;
      
   &lt;/head&gt;
   
   &lt;body onload = "loadMap()"&gt;
