        <hr> 
        <p>R packages are a collection of R functions, complied code and sample data. They are stored under a directory called <b>"library"</b> in the R environment. By default, R installs a set of packages during installation. More packages are added later, when they are needed for some specific purpose. When we start the R console, only the default packages are available by default. Other packages which are already installed have to be loaded explicitly to be used by the R program that is going to use them.</p> 
        <p>All the packages available in R language are listed at <a href="https://cran.r-project.org/web/packages/available_packages_by_name.html" target="_blank" rel="nofollow">R Packages.</a></p> 
        <p>Below is a list of commands to be used to check, verify and use the R packages.</p> 
        <h2>Check Available R Packages</h2> 
        <p>Get library locations containing R packages</p> 
        <pre class="prettyprint notranslate tryit">
.libPaths()
</pre> 
        <p>When we execute the above code, it produces the following result. It may vary depending on the local settings of your pc.</p> 
        <pre class="result notranslate">
[2] "C:/Program Files/R/R-3.2.2/library"
</pre> 
        <h2>Get the list of all the packages installed</h2> 
        <pre class="prettyprint notranslate tryit">
library()
</pre> 
        <p>When we execute the above code, it produces the following result. It may vary depending on the local settings of your pc.</p> 
        <pre class="result notranslate">
Packages in library ‘C:/Program Files/R/R-3.2.2/library’:

base                    The R Base Package
boot                    Bootstrap Functions (Originally by Angelo Canty
                        for S)
class                   Functions for Classification
cluster                 "Finding Groups in Data": Cluster Analysis
                        Extended Rousseeuw et al.
codetools               Code Analysis Tools for R
compiler                The R Compiler Package
</pre> 
        <p>Get all packages currently loaded in the R environment</p> 
        <pre class="prettyprint notranslate tryit">
search()
</pre> 
        <p>When we execute the above code, it produces the following result. It may vary depending on the local settings of your pc.</p> 
        <pre class="result notranslate">
[1] ".GlobalEnv"        "package:stats"     "package:graphics" 
[4] "package:grDevices" "package:utils"     "package:datasets" 
[7] "package:methods"   "Autoloads"         "package:base" 
</pre> 
        <h2>Install a New Package</h2> 
        <p>There are two ways to add new R packages. One is installing directly from the CRAN directory and another is downloading the package to your local system and installing it manually.</p> 
        <h3>Install directly from CRAN</h3> 
        <p>The following command gets the packages directly from CRAN webpage and installs the package in the R environment. You may be prompted to choose a nearest mirror. Choose the one appropriate to your location.</p> 
        <pre class="result notranslate">
 install.packages("Package Name")
 
# Install the package named "XML".
 install.packages("XML")
</pre> 
        <h3>Install package manually</h3> 
        <p>Go to the link <a target="_blank" rel="nofollow" href="https://cran.r-project.org/web/packages/available_packages_by_name.html">R Packages</a> to download the package needed. Save the package as a <b>.zip</b> file in a suitable location in the local system.</p> 
        <p>Now you can run the following command to install this package in the R environment.</p> 
        <pre class="prettyprint notranslate">
install.packages(file_name_with_path, repos = NULL, type = "source")

# Install the package named "XML"
install.packages("E:/XML_3.98-1.3.zip", repos = NULL, type = "source")
</pre> 
        <h2>Load Package to Library</h2> 
        <p>Before a package can be used in the code, it must be loaded to the current R environment. You also need to load a package that is already installed previously but not available in the current environment.</p> 
        <p>A package is loaded using the following command −</p> 
        <pre class="result notranslate">
library("package Name", lib.loc = "path to library")

# Load the package named "XML"
install.packages("E:/XML_3.98-1.3.zip", repos = NULL, type = "source")
</pre>
