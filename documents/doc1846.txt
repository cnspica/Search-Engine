Especially, because the years here gave me a lot of inspiration and&nbsp;enlightenment.&lt;/p&gt; &lt;p&gt;The book&nbsp;is a collection of useful and interesting LaTeX recipes. I hope you will enjoy it.&lt;/p&gt; &lt;p&gt;The publisher agreed with me to give 35% discount to LaTeX friends who pre-order the book. This is valid until October 18th, 2015.    <content:encoded>
    &lt;p&gt;&lt;a href="http://tex.blogoverflow.com/files/2015/06/Screen-Shot-2015-06-05-at-23.08.04-.png"&gt;&lt;img src="http://tex.blogoverflow.com/files/2015/06/Screen-Shot-2015-06-05-at-23.08.04--300x90.png" alt="Screen Shot 2015-06-05 at 23.08.04" width="300" height="90" class="alignnone size-medium wp-image-2385" /&gt;&lt;/a&gt;&lt;/p&gt; &lt;p&gt;Yesterday, we lost Hermann Zapf. Brilliant type designer, a very special person. Hermann was 96.&lt;/p&gt; &lt;p&gt;He lived in Darmstadt, Germany and had been an Honorary member of the board of the TeX Users Group since 1980, with the deserved title of “Wizard of Fonts”.&lt;/p&gt; &lt;p&gt;Hermann Zapf directly contributed to the TeX world with the Euler fonts and with several discussions with Donald Knuth about the design of the Computer Modern fonts. His legacy to the world of typography and type design is too big to be reduced to a list of his contributions: just the mention of Palatino, Optima and Zapfino, three fonts he designed, can illustrate his craftsmanship in type design.&lt;/p&gt; &lt;p&gt;Hermann was also a precursor of micro-typography in computer aided typesetting, with his HZ system on which the micro-typographic features of pdfTeX, XeTeX and LuaTeX are based.&lt;/p&gt; &lt;p&gt;Thank you for everything, Hermann.&lt;/p&gt; &lt;div id="attachment_1954" style="width: 417px" class="wp-caption aligncenter"&gt;&lt;a href="http://tex.blogoverflow.com/files/2012/11/hermann.jpg"&gt;&lt;img class="size-full wp-image-1954" src="http://tex.blogoverflow.com/files/2012/11/hermann.jpg" alt="" width="407" height="500" /&gt;&lt;/a&gt;&lt;p class="wp-caption-text"&gt;Hermann Zapf at EuroTeX 2005 in Pont-a-Mousson, France (by &lt;a href="http://www.flickriver.com/photos/adamt/290946217/"&gt;Adam Twardoch&lt;/a&gt;).&lt;/p&gt;&lt;/div&gt;

    &lt;p&gt;In agreement with the publisher, the &amp;#8220;&lt;a href="http://link.packtpub.com/lOKssB" title="Publisher's page: LaTeX Beginner's Guide"&gt;LaTeX Beginner&amp;#8217;s Guide&lt;/a&gt;&amp;#8221; ebook will soon be officially available for free download. &lt;/ul&gt; &lt;h1&gt;Folder Structure&lt;/h1&gt; &lt;p&gt;The template uses a recommended folder structure which looks like: &lt;pre&gt;&amp;gt; 00_Backup &amp;gt; 01_Preamble &amp;gt; 02_Chapters &amp;gt; It includes different tex files using the &lt;strong&gt;\input&lt;/strong&gt; command which leads to a very tidy and readable structure. &lt;pre&gt;%%% File encoding is ISO-8859-1 (also known as Latin-1) %%% You can use special characters just liek ä,ü and ñ %%% LaTeX template by Manuel Kuehner, 2015&lt;/pre&gt; &lt;h1&gt;Changing the Two Main Layout Colors&lt;/h1&gt; &lt;p&gt;The template uses two colors as eye candy. There is a video tutorial that shows how to setup up such a system:&lt;/p&gt; &lt;p&gt;&lt;strong&gt;YouTube:&lt;/strong&gt;&nbsp;&lt;a href="https://www.youtube.com/watch?v=FxKtwdob2RQ"&gt;Installing LaTeX/MiKTeX and Texmaker on a Windows 7 System (Version 2014)&lt;/a&gt;&lt;/p&gt; &lt;h1&gt;Batch Files&lt;/h1&gt; &lt;p&gt;I added two batch files to the template which work under Windows in combination with &lt;a title="MiKTeX…typesetting beautiful documents…" href="http://miktex.org/" target="_blank"&gt;MiKTeX&lt;/a&gt;.&lt;/p&gt; &lt;ol&gt; &lt;li&gt;The first batch file is named&nbsp;&lt;strong&gt;Batch_CleanUp.bat&lt;/strong&gt; and deletes all auxiliary files like .aux, .log and so on.&lt;/li&gt; &lt;li&gt;The second batch file is named&nbsp;&lt;strong&gt;Batch_CompileFile.bat&lt;/strong&gt; and calls&nbsp;&lt;strong&gt;Batch_CleanUp.bat&lt;/strong&gt;, &lt;strong&gt;pdflatex&lt;/strong&gt; and &lt;strong&gt;biber&lt;/strong&gt;.&nbsp;&lt;strong&gt;pdflatex&lt;/strong&gt; is executed with some additional options which are useful when you have a large document and use the external option of &lt;strong&gt;TikZ/PGF&lt;/strong&gt;.&lt;/li&gt; &lt;/ol&gt; &lt;h2&gt;Batch_CleanUp.bat&lt;/h2&gt; &lt;p&gt;&lt;pre&gt;:: Delete all auxiliary files del *.aux del *.log del *.gz del *.blg del *.bbl del *.lof del *.lot del *.out del *.ptc del *.toc del *blx.bib del *run.xml del *.bcf &lt;/pre&gt;&lt;/p&gt; &lt;h2&gt;Batch_CompileFile.bat&lt;/h2&gt; &lt;p&gt;&lt;pre&gt;:: Delete all auxiliary files rem --- 'rem' or '::' is the key word for comments in batch files like this&lt;/p&gt; &lt;p&gt;rem --- Call cleanup script in order to delete all auxiliary files --- rem --- Sometimes after error messages there are still errors in the auxiliary files even if the root of the error was corrected in the tex file :: --- call my_Batch_CleanUp.bat :: ---&lt;/p&gt; &lt;p&gt;rem --- First LaTeX run (pdflatex) with special options (more memory) for large LaTeX projects and tikz stuff (-shell-escape) --- rem --- I use MiKTeX -- maybe you need another syntax if you use TeX Live for example --- :: --- pdflatex --extra-mem-top=60000000 -synctex=1 -shell-escape -interaction=batchmode MainFile.tex&lt;/p&gt; &lt;p&gt;rem --- Execute biber if used --- :: --- biber MainFile :: ---&lt;/p&gt; &lt;p&gt;rem --- Compile three times to make sure everything is correct --- :: --- pdflatex --extra-mem-top=60000000 -synctex=1 -shell-escape -interaction=batchmode MainFile.tex pdflatex --extra-mem-top=60000000 -synctex=1 -shell-escape -interaction=batchmode MainFile.tex pdflatex --extra-mem-top=60000000 -synctex=1 -shell-escape -interaction=batchmode MainFile.tex :: --- &lt;/pre&gt;&lt;/p&gt;
