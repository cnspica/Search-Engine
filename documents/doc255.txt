        <p>In this chapter, we will study about <b>Creating Modules</b> in Joomla. Modules are the extensions which are flexible and lightweight and useful for page rendering.</p> 
        <h2>Create Modules</h2> 
        <p>Following are the simple steps to create modules in Joomla.</p> 
        <p><b>Step (1)</b> − Create a folder called <b>mod_firstmodule</b> in your <b>Joomla</b> → <b>modules</b> folder.</p> 
        <img src="/joomla/images/joomla-createmodules/joomla-createmodules-step1.jpg" alt="Joomla Create Modules"> 
        <p><b>Step (2)</b> − In the <b>mod_firstmodule</b> folder create a file called as "helper.php". This file contains class name as helper, it helps to display the retrieved data in the module output.</p> 
        <h3>helper.php</h3> 
        <pre class="prettyprint notranslate">
&lt;?php
   /**
      * Helper class for Hello World! module
      *
      * @package    Joomla.Tutorials
      * @subpackage Modules
      * @link http://docs.joomla.org/J3.x:Creating_a_simple_module/Developing_a_Basic_Module
      * @license        GNU/GPL, see LICENSE.php
      * mod_helloworld is free software. This version may have been modified pursuant
      * to the GNU General Public License, and as distributed it includes or
      * is derivative of works licensed under the GNU General Public License or
      * other free or open source software licenses.
   */
		
   class ModHelloWorldHelper {
      /**
         * Retrieves the hello message
         *
         * @param   array  $params An object containing the module parameters
         *
        * @access public
      */
		
      public static function getHello($params) {
         return 'Hello, World!';
      }
   }
	
?&gt;
</pre> 
        <p><b>Step (3)</b> − Create a file called as <b>mod_helloworld.php</b>. It is an entry point for the module which performs initialization routines, collects necessary data and displays the module output using template.</p> 
        <h3>mod_helloworld.php</h3> 
        <pre class="prettyprint notranslate">
&lt;?php
   /**
      * Hello World! Module Entry Point
      *
      * @package    Joomla.Tutorials
      * @subpackage Modules
      * @license    GNU/GPL, see LICENSE.php
      * @link       http://docs.joomla.org/J3.x:Creating_a_simple_module/Developing_a_Basic_Module
      * mod_helloworld is free software. This version may have been modified pursuant
      * to the GNU General Public License, and as distributed it includes or
      * is derivative of works licensed under the GNU General Public License or
      * other free or open source software licenses.
   */

   // No direct access
   defined('_JEXEC') or die;

   // Include the syndicate functions only once
   require_once dirname(__FILE__) . '/helper.php';

   $hello = modHelloWorldHelper::getHello($params);
   require JModuleHelper::getLayoutPath('mod_helloworld');
?&gt;
</pre> 
        <p><b>Step (4)</b> − Create a <b>mod_helloworld.xml file</b>. This file contains the information about module. This xml file contains information of files that are to be installed in Joomla for the module.</p> 
        <h3>mod_helloworld.xml file</h3> 
        <pre class="prettyprint notranslate">
&lt;?xml version = "1.0" encoding = "utf-8"?&gt;

&lt;extension type = "module" version = "3.1.0" client = "site" method="upgrade"&gt;

   &lt;name&gt;Hello, World!&lt;/name&gt;
   &lt;author&gt;Tutorials Point&lt;/author&gt;
   &lt;version&gt;1.0.0&lt;/version&gt;
   &lt;description&gt;A simple Hello World! module.&lt;/description&gt;
	
   &lt;files&gt;
      &lt;filename&gt;mod_helloworld.xml&lt;/filename&gt;
      &lt;filename module = "mod_helloworld"&gt;mod_helloworld.php&lt;/filename&gt;
      &lt;filename&gt;index.html&lt;/filename&gt;
      &lt;filename&gt;helper.php&lt;/filename&gt;
      &lt;filename&gt;tmpl/default.php&lt;/filename&gt;
      &lt;filename&gt;tmpl/index.html&lt;/filename&gt;
   &lt;/files&gt;
	
   &lt;config&gt;
   &lt;/config&gt;
	
&lt;/extension&gt;
</pre> 
        <p><b>Step (5)</b> − Create a simple html file called <b>index.html</b>. The purpose of writing this file is that, the created directories should not be browsed. When a user browses into these directories, the index.html file gets displayed. You can even keep this file empty.</p> 
        <h3>index.html</h3> 
        <pre class="prettyprint notranslate">
&lt;html&gt;
   &lt;body&gt; Welcome to Tutorials Point!!!!! &lt;/body&gt;
&lt;/html&gt;
</pre> 
        <p><b>Step (6)</b> − Create a folder called as <b>tmpl</b>. Place <b>default.php</b> file as shown below and index.html (created in step (5)) under <b>tmpl</b> folder. The default.php file is a template that displays the module output.</p> 
        <h3>default.php</h3> 
        <pre class="prettyprint notranslate">
&lt;?php
   /**
      * @package Joomla.Site
      * @subpackage mod_firstmodule
      * @copyright Copyright (C) 2005 - 2012 Open Source Matters, Inc. All rights reserved.
      * @license GNU General Public License version 2 or later; see LICENSE.txt
   */
 
defined('_JEXEC') or die;
&gt;

&lt;p&gt;Hello World!!!!!!&lt;/p&gt;
</pre> 
        <p>After you have finished creating all these files, compress the complete folder <b>mod_firstmodule</b>.</p> 
        <p><b>Step (7)</b> − Go to <b>Extension</b> → <b>Extension Manager</b> in Joomla administrator and you will get the following screen. Here you can upload and install your created module files i.e. <b>mod_firstmodule</b> folder. Click on <b>Choose File</b> and select the created module file (compressed one). Click on <b>Upload &amp; Install</b> button to upload the module file.</p> 
        <img src="/joomla/images/joomla-createmodules/joomla-createmodules-step7.jpg" alt="Joomla Create Modules"> 
        <p><b>Step (8)</b> − After upload and installation, go to <b>Module Manager</b> and click on <b>New</b>. There you can view your created module file as shown below.</p> 
        <img src="/joomla/images/joomla-createmodules/joomla-createmodules-step8.jpg" alt="Joomla Create Modules"> 
        <p><b>Step (9)</b> − You can assign this module similar to the other modules and then publish it.</p>
