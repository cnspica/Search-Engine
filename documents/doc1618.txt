        <h1>Google Maps - UI Controls</h1>

        <p>Google Maps provides a User Interface with various controls to let the user interact with the map. We can add, customize, and disable these controls.</p> 
        <h2>Default Controls</h2> 
        <p>Here is a list of the default controls provided by Google Maps −</p> 
        <ul class="list"> 
         <li><p><b>Zoom</b> − To increase and decease the zoom level of the map, we will have a slider with + and − buttons, by default. This slider will be located at the corner of left hand side of the map.</p></li> 
         <li><p><b>Pan</b> − Just above the zoom slider, there will be a pan control for panning the map.</p></li> 
         <li><p><b>Map Type</b> − You can locate this control at the top right corner of the map. It provides map type options such as Satellite, Roadmap, and Terrain. Users can choose any of these maps.</p></li> 
         <li><p><b>Street view</b> − Between the pan icon and the zoom slider, we have a pegman icon. Users can drag this icon and place at a particular location to get its street view.</p></li> 
        </ul> 
        <h3>Example</h3> 
        <p>Here is an example where you can observe the default UI controls provided by Google Maps −</p> 
        <iframe class="result" src="/google_maps/src/map_with_default_controls.htm" height="420px" width="600px"></iframe> 
        <h2>Disabling the UI Default Controls</h2> 
        <p>We can disable the default UI controls provided by Google Maps simply by making the <b>disableDefaultUI</b> value true in the map options.</p> 
        <h3>Example</h3> 
        <p>The following example shows how to disable the default UI controls provided by Google Maps.</p> 
        <pre class="prettyprint notranslate tryit">
&lt;!DOCTYPE html&gt;
&lt;html&gt;
   
   &lt;head&gt;
      &lt;script src = "http://maps.googleapis.com/maps/api/js"&gt;&lt;/script&gt;
      
      &lt;script&gt;
         function loadMap() {
			
            var mapOptions = {
               center:new google.maps.LatLng(17.609993, 83.221436),
               zoom:5,
               mapTypeId:google.maps.MapTypeId.ROADMAP,
               disableDefaultUI: true
            };
				
            var map = new google.maps.Map(document.getElementById("sample"),mapOptions);
         }
      &lt;/script&gt;
      
   &lt;/head&gt;
   
   &lt;body onload = "loadMap()"&gt;
      &lt;div id = "sample" style = "width:580px; height:400px;"&gt;&lt;/div&gt;
   &lt;/body&gt;

&lt;/html&gt;
</pre> 
        <p>It will produce the following output −</p> 
        <iframe class="result" src="/google_maps/src/disabling_default_controls.htm" height="420px" width="600px"></iframe> 
        <h2>Enabling/Disabling All the Controls</h2> 
        <p>In addition to these default controls, Google Maps also provides three more controls as listed below.</p> 
        <ul class="list"> 
         <li><p><b>Scale</b> − The Scale control displays a map scale element. This control is not enabled by default.</p></li> 
         <li><p><b>Rotate</b> − The Rotate control contains a small circular icon which allows you to rotate maps containing oblique imagery. This control appears by default at the top left corner of the map. (See 45° Imagery for more information.)</p></li> 
         <li><p><b>Overview</b> − To increase and decease the zoom level of the map, we have a slider with + and − buttons, by default. This slider is located at the left corner of the map.</p></li> 
        </ul> 
        <p>In the map options, we can enable and disable any of the controls provided by Google Maps as shown below −</p> 
        <pre class="prettyprint notranslate">
{
   panControl: boolean,
   zoomControl: boolean,
   mapTypeControl: boolean,
   scaleControl: boolean,
   streetViewControl: boolean,
   overviewMapControl: boolean
}
</pre> 
        <h3>Example</h3> 
        <p>The following code shows how to enable all the controls −</p> 
        <pre class="prettyprint notranslate tryit">
&lt;!DOCTYPE html&gt;
&lt;html&gt;
   
   &lt;head&gt;
      &lt;script src = "http://maps.googleapis.com/maps/api/js"&gt;&lt;/script&gt;
      
      &lt;script&gt;
         function loadMap() {
			
            var mapOptions = {
               center:new google.maps.LatLng(19.373341, 78.662109),
               zoom:5,
               panControl: true,
               zoomControl: true,
               scaleControl: true,
               mapTypeControl:true,
               streetViewControl:true,
               overviewMapControl:true,
               rotateControl:true
            }
				
            var map = new google.maps.Map(document.getElementById("sample"),mapOptions);
         }
      &lt;/script&gt;
      
   &lt;/head&gt;
   
   &lt;body onload = "loadMap()"&gt;
      &lt;div id = "sample" style = "width:580px; height:400px;"&gt;&lt;/div&gt;
   &lt;/body&gt;

&lt;/html&gt;
</pre> 
        <p>It will produce the following output −</p> 
        <iframe class="result" src="/google_maps/src/enabling_all_controls.htm" height="420px" width="600px"></iframe> 
        <h2>Control Options</h2> 
        <p>We can change the appearance of Google Maps controls using its control options. For example, the zoom control can be either reduced or enlarged in size. The MapType control appearance can be varied to a horizontal bar or a drop-down menu. Given below is a list of Control options for Zoom and MapType controls.</p> 
        <table class="table table-bordered"> 
         <tbody>
          <tr> 
           <th>Sr.No.</th> 
           <th>Control Name</th> 
           <th>Control Options</th> 
          </tr> 
          <tr> 
           <td>1</td> 
           <td>Zoom control</td> 
           <td> 
            <ul class="list"> 
             <li>google.maps.ZoomControlStyle.SMALL</li> 
             <li>google.maps.ZoomControlStyle.LARGE</li> 
             <li>google.maps.ZoomControlStyle.DEFAULT</li> 
            </ul> </td> 
          </tr> 
          <tr> 
           <td>2</td> 
           <td>MapType control</td> 
           <td> 
            <ul class="list"> 
             <li>google.maps.MapTypeControlStyle.HORIZONTAL_BAR</li> 
             <li>google.maps.MapTypeControlStyle.DROPDOWN_MENU</li> 
             <li>google.maps.MapTypeControlStyle.DEFAULT</li> 
            </ul> </td> 
          </tr> 
         </tbody>
        </table> 
        <h3>Example</h3> 
        <p>The following example demonstrates how to use the control options −</p> 
        <pre class="prettyprint notranslate tryit">
&lt;!DOCTYPE html&gt;
&lt;html&gt;
   
   &lt;head&gt;
      &lt;script src = "http://maps.googleapis.com/maps/api/js"&gt;&lt;/script&gt;
      
      &lt;script&gt;
         function loadMap() {
			
            var mapOptions = {
               center:new google.maps.LatLng(19.373341, 78.662109),
               zoom:5,
               mapTypeControl: true,
               
               mapTypeControlOptions: {
                  style: google.maps.MapTypeControlStyle.DROPDOWN_MENU, mapTypeIds: [
                     google.maps.MapTypeId.ROADMAP,
                     google.maps.MapTypeId.TERRAIN
                  ]
               },
					
               zoomControl: true,
               
               zoomControlOptions: {
                  style: google.maps.ZoomControlStyle.SMALL
               }
            }
				
            var map = new google.maps.Map(document.getElementById("sample"),mapOptions);
         }
      &lt;/script&gt;
      
   &lt;/head&gt;
   
   &lt;body onload = "loadMap()"&gt;
      &lt;div id = "sample" style = "width:580px; height:400px;"&gt;&lt;/div&gt;
   &lt;/body&gt;
   
&lt;/html&gt;
</pre> 
        <p>It produces the following output −</p> 
        <iframe class="result" src="/google_maps/src/control_options.htm" height="420px" width="600px"></iframe> 
        <h2>Control Positioning</h2> 
        <p>You can change the position of the controls by adding the following line in the control options.</p> 
        <pre class="result notranslate">
position:google.maps.ControlPosition.<b>Desired_Position</b>,
</pre> 
        <p>Here is the list of available positions where a control can be placed on a map −</p> 
        <ul class="list"> 
         <li>TOP_CENTER</li> 
         <li>TOP_LEFT</li> 
         <li>TOP_RIGHT</li> 
         <li>LEFT_TOP</li> 
         <li>RIGHT_TOP</li> 
         <li>LEFT_CENTER</li> 
         <li>RIGHT_CENTER</li> 
         <li>LEFT_BOTTOM</li> 
         <li>RIGHT_BOTTOM</li> 
         <li>BOTTOM_CENTER</li> 
         <li>BOTTOM_LEFT</li> 
         <li>BOTTOM_RIGHT</li> 
        </ul> 
        <h3>Example</h3> 
        <p>The following example shows how to place the MapTypeid control at the top centre of the map and how to place the zoom control at the bottom centre of the map.</p> 
        <pre class="prettyprint notranslate tryit">
&lt;!DOCTYPE html&gt;
&lt;html&gt;
   
   &lt;head&gt;
      &lt;script src = "http://maps.googleapis.com/maps/api/js"&gt;&lt;/script&gt;
      
      &lt;script&gt;
         function loadMap() {
			
            var mapOptions = {
               center:new google.maps.LatLng(19.373341, 78.662109),
               zoom:5,
               mapTypeControl: true,
               
               mapTypeControlOptions: {
                  style: google.maps.MapTypeControlStyle.DROPDOWN_MENU,
                  position:google.maps.ControlPosition.TOP_CENTER,
                  
                  mapTypeIds: [
                     google.maps.MapTypeId.ROADMAP,
                     google.maps.MapTypeId.TERRAIN
                  ]
               },
					
               zoomControl: true,
               
               zoomControlOptions: {
                  style: google.maps.ZoomControlStyle.SMALL,
                  position:google.maps.ControlPosition.BOTTOM_CENTER
               }
            }
				
            var map = new google.maps.Map(document.getElementById("sample"),mapOptions);
         }
      &lt;/script&gt;
      
   &lt;/head&gt;
   
   &lt;body onload = "loadMap()"&gt;
      &lt;div id = "sample" style = "width:580px; height:400px;"&gt;&lt;/div&gt;
   &lt;/body&gt;
   
&lt;/html&gt; 
</pre>
