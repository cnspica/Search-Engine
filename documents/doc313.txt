        <hr> 
        <p>In this chapter, we will study about <b>Latest News Module</b> in Joomla. Latest News Module is used to display the most recent published article in a list.</p> 
        <h2>Latest News Module</h2> 
        <p>Following are the simple steps to edit an existing Latest News Module in Joomla.</p> 
        <p><b>Step (1)</b> − Click <b>Extension</b> → <b>Module Manager</b> in Joomla administrator control panel. Following screen will be displayed −</p> 
        <img src="/joomla/images/joomla-latestnewsmodule/joomla-latestnewsmodule-step1.jpg" alt="Joomla Latest News Module"> 
        <p><b>Step (2)</b> − Click on <b>New</b> button, you get a screen as shown below. Now select Module Type <i>Latest News</i>.</p> 
        <img src="/joomla/images/joomla-latestnewsmodule/joomla-latestnewsmodule-step2.jpg" alt="Joomla Latest News Module"> 
        <p><b>Step (3)</b> − After clicking on <b>Latest News</b>, screen displayed is as shown below.</p> 
        <img src="/joomla/images/joomla-latestnewsmodule/joomla-latestnewsmodule-step3.jpg" alt="Joomla Latest News Module"> 
        <p>By default, the <b>Module</b> tab is displayed. The Module tab displays the Latest News.</p> 
        <p>Below, we have mentioned the details of the fields present in the Module Latest News page.</p> 
        <ul class="list"> 
         <li><p><b>Title</b> − Displays the title of Module.</p></li> 
         <li><p><b>Category</b> − Selects category for the article to display.</p></li> 
         <li><p><b>Count</b> − Specifies the number of articles to be displayed.</p></li> 
         <li><p><b>Featured Articles</b> − <i>(Show/Hide/Only show Featured Articles)</i> The articles can be displayed, hidden or you can only display the featured articles on the front page.</p></li> 
         <li><p><b>Order</b> − <i>(Recently Added First/Recently Modified First/Recently Published First/Recently Touched First/Random Articles)</i> Sets the order of the articles depending on recent activities done for the article.</p></li> 
         <li><p><b>Authors</b> − <i>(Anyone/Added or modified by me/Not added or modified by me)</i> Displays the article based on the author who has added or modified it.</p></li> 
         <li><p><b>Show Title</b> − Displays the title of the module in the front end.</p></li> 
         <li><p><b>Position</b> − It selects the position of the module to be displayed.</p></li> 
         <li><p><b>Status</b> − It displays the status of the article such as <i>Published, Unpublished</i> and <i>Trashed</i>.</p></li> 
         <li><p><b>Start Publishing</b> − It starts publishing on the assigned date and time.</p></li> 
         <li><p><b>Finish Publishing</b> − It finishes publishing on the assigned date and time.</p></li> 
         <li><p><b>Access</b> −It allows only the selected users from the dropdown list such as <i>Guest, Public, Super Users, Registered</i> and <i>Special options</i> to view the items.</p></li> 
         <li><p><b>Ordering</b> − Displays the dropdown of all modules present in the position.</p></li> 
         <li><p><b>Language</b> − It specifies the language of the site. It provides either <i>ALL</i> or <i>English (UK)</i> options.</p></li> 
         <li><p><b>Note</b> − It is used to enter the text information.</p></li> 
        </ul> 
        <p><b>Step (4)</b> − Click on <b>Menu Assignment</b> tab, you will get the screen as shown below.</p> 
        <img src="/joomla/images/joomla-latestnewsmodule/joomla-latestnewsmodule-step4.jpg" alt="Joomla Latest News Module"> 
        <ul class="list"> 
         <li><p><b>Module Assignment</b> − Select the option to view the item from the dropdown list such as <i>on all pages, no pages, only on the pages selected</i> or <i>on all pages except those selected</i>.</p></li> 
         <li><p><b>Menu Selection</b> − After selecting the option <b>only on the pages selected</b> or <b>on all pages except those selected</b>, it displays all the menu items present in Joomla. It allows you to assign module to some and not all pages.</p></li> 
        </ul> 
        <p><b>Step (5)</b> − <b> Module Permissions</b> tab offers three actions to choose from. It manages the setting of module permission for the user groups as shown in the following screen.</p> 
        <img src="/joomla/images/joomla-latestnewsmodule/joomla-latestnewsmodule-step5.jpg" alt="Joomla Latest News Module"> 
        <ul class="list"> 
         <li><p><b>Delete</b> − It allows users in group to delete the content present in the module.</p></li> 
         <li><p><b>Edit</b> − It allows users in group to edit the content present in the module.</p></li> 
         <li><p><b>Edit State</b> − It allows users in group to change the state of content present in the module.</p></li> 
        </ul> 
        <p><b>Step (6)</b> − The <b>Advanced</b> tab is used to do the advanced settings of the latest news. The advanced tab screen is shown below.</p> 
        <img src="/joomla/images/joomla-latestnewsmodule/joomla-latestnewsmodule-step6.jpg" alt="Joomla Latest News Module"> 
        <ul class="list"> 
         <li><p><b>Alternative Layout</b> − Select the layout to be used for this module, which you have already defined as one or more layouts for a module in Joomla or template.</p></li> 
         <li><p><b>Module Class Suffix</b> − Setting this parameter causes Joomla to either add a new CSS class or modify the existing CSS class for the div element for this specific module.</p></li> 
         <li><p><b>Caching</b> − Cache the content of the module. <b>Use Global</b> is used to cache settings from global configuration. <b>No Caching</b> is used whether to cache the content of the module or not.</p></li> 
         <li><p><b>Cache Time</b> − Specify the time in minutes before the module is re-cached.</p></li> 
         <li><p><b>Module Tag</b> − Specify the HTML tag for the modules. </p></li> 
         <li><p><b>Bootstrap Size</b> − Select width of the module built into bootstrap.</p></li> 
         <li><p><b>Header Tag</b> − Modules uses the HTML header tags.</p></li> 
         <li><p><b>Header Class</b> − Optional CSS classes are added in the modules header.</p></li> 
         <li><p><b>Module Style</b> − Overrides the templates style for its position.</p></li> 
        </ul> 
        <h2>Toolbar</h2> 
        <p>Following are the details about the toolbar function tabs present in Latest News Module.</p> 
        <ul class="list"> 
         <li><p><b>Save</b> − Saves your modules.</p></li> 
         <li><p><b>Save &amp; Close</b> − Saves the modules and will close the current screen.</p></li> 
         <li><p><b>Save &amp; New</b> − Saves the modules and opens a new creating module screen.</p></li> 
         <li><p><b>Cancel</b> − Cancels the created modules in Joomla.</p></li> 
        </ul>
