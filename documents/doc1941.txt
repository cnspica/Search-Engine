            <div class="post-text" itemprop="text"> 
             <p>You can install Google Chrome in Ubuntu this way - open a terminal and execute these commands: </p> 
             <pre><code>echo "deb http://dl.google.com/linux/chrome/deb/ stable main" | sudo tee /etc/apt/sources.list.d/google-chrome.list
wget -q -O - https://dl.google.com/linux/linux_signing_key.pub | sudo apt-key add -
sudo apt-get update
sudo apt install google-chrome-stable  
</code></pre> 
             <p>In case you receive an <code>apt-get update</code> error, please read : <a href="http://askubuntu.com/questions/724093/no-more-updates-for-google-chrome-apt-get-update-error/724170#724170">Google Chrome | apt-get update error</a></p>

            <div class="post-text" itemprop="text"> 
             <p>You can go on the Google website and download the .deb package from this url : <a href="https://www.google.fr/chrome/browser/desktop/index.html" rel="nofollow">https://www.google.fr/chrome/browser/desktop/index.html</a></p> 
             <p>The google website will auto-detect you're operating system here GNU/Linux, if not click on other platform and select "Linux" if will give you two options an RPM and a DEB package, as you run Ubuntu, you will choose the "64 bit .deb (For Debian/Ubuntu)" package.</p> 
             <p>Once downloaded you just have to run the command </p> 
             <pre><code>sudo dpkg --install /path/to/package.deb
</code></pre> 
             <p>If will say that it miss some dependencies which you can fix by running this command </p> 
             <pre><code>sudo apt-get install --force
sudo dpkg --install /path/to/package.deb
</code></pre>
