        <p>The Scripting Layer for Android (SL4A) is a library that helps write scripts targeting the Android Platform. SL4A supports a number of scripting languages like Perl, Python, JRuby, PHP etc. It can also be extended to support new languages. While Java can be used for Android Development, it is a good thing when all you need to do is write a simple test case or turn on or off settings on your device. This tutorial will help you automate your Android device using simple Python scripts.</p> 
        <h1>Audience</h1> 
        <p>Android is the monopoly of Java enthusiasts. Well, if you are not a Java admirer and wondering, how you can develop Android Apps, here is something for your benefit. This tutorial introduces you to developing Android Applications using Python.</p> 
        <h1>Prerequisites</h1> 
        <p>Before you start proceeding with this tutorial, we are assuming that you are already aware about the Python. If you are not well aware of these concepts, then we will suggest you to go through our short tutorials on Python.</p>
