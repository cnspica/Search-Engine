        <hr> 
        <p>Data Reshaping in R is about changing the way data is organized into rows and columns. Most of the time data processing in R is done by taking the input data as a data frame. It is easy to extract data from the rows and columns of a data frame but there are situations when we need the data frame in a format that is different from format in which we received it. R has many functions to split, merge and change the rows to columns and vice-versa in a data frame.</p> 
        <h2>Joining Columns and Rows in a Data Frame</h2> 
        <p>We can join multiple vectors to create a data frame using the <b>cbind()</b>function. Also we can merge two data frames using <b>rbind()</b> function.</p> 
        <pre class="prettyprint notranslate tryit">
# Create vector objects.
city &lt;- c("Tampa","Seattle","Hartford","Denver")
state &lt;- c("FL","WA","CT","CO")
zipcode &lt;- c(33602,98104,06161,80294)

# Combine above three vectors into one data frame.
addresses &lt;- cbind(city,state,zipcode)

# Print a header.
cat("# # # # The First data frame\n") 

# Print the data frame.
print(addresses)

# Create another data frame with similar columns
new.address &lt;- data.frame(
   city = c("Lowry","Charlotte"),
   state = c("CO","FL"),
   zipcode = c("80230","33949"),
   stringsAsFactors = FALSE
)

# Print a header.
cat("# # # The Second data frame\n") 

# Print the data frame.
print(new.address)

# Combine rows form both the data frames.
all.addresses &lt;- rbind(addresses,new.address)

# Print a header.
cat("# # # The combined data frame\n") 

# Print the result.
print(all.addresses)
</pre>

        <h2>Merging Data Frames</h2> 
        <p>We can merge two data frames by using the <b>merge()</b> function. The data frames must have same column names on which the merging happens.</p> 
        <p>In the example below, we consider the data sets about Diabetes in Pima Indian Women available in the library names "MASS". we merge the two data sets based on the values of blood pressure("bp") and body mass index("bmi"). On choosing these two columns for merging, the records where values of these two variables match in both data sets are combined together to form a single data frame.</p> 
        <pre class="prettyprint notranslate tryit">
library(MASS)
merged.Pima &lt;- merge(x = Pima.te, y = Pima.tr,
   by.x = c("bp", "bmi"),
   by.y = c("bp", "bmi")
)
print(merged.Pima)
nrow(merged.Pima)
</pre>

        <h2>Melting and Casting</h2> 
        <p>One of the most interesting aspects of R programming is about changing the shape of the data in multiple steps to get a desired shape. The functions used to do this are called <b>melt()</b> and <b>cast()</b>.</p> 
        <p>We consider the dataset called ships present in the library called "MASS".</p> 
        <pre class="prettyprint notranslate tryit">
library(MASS)
print(ships)
</pre>

        <h2>Melt the Data</h2> 
        <p>Now we melt the data to organize it, converting all columns other than type and year into multiple rows.</p> 
        <pre class="prettyprint notranslate tryit">
molten.ships &lt;- melt(ships, id = c("type","year"))
print(molten.ships)
</pre>

        <h2>Cast the Molten Data</h2> 
        <p>We can cast the molten data into a new form where the aggregate of each type of ship for each year is created. It is done using the <b>cast()</b> function.</p> 
        <pre class="prettyprint notranslate tryit">
recasted.ship &lt;- cast(molten.ships, type+year~variable,sum)
print(recasted.ship)
</pre>
