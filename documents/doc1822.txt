      <h1 class="blog-title"><a href="http://blog.superuser.com/2013/10/23/what-to-do-after-buying-a-new-laptop/">What to Do After Buying a New Laptop</a></h1> 
      <div class="meta-info">
       <span class="postedDate">October 23, 2013</span> by 
       <a href="http://blog.superuser.com/author/JTRipton/" title="Posts by JTRipton" rel="author">JTRipton</a>. 
       <a href="http://blog.superuser.com/2013/10/23/what-to-do-after-buying-a-new-laptop/#comments">11 comments</a> 
      </div> 
      <div class="post-text"> 
       <p>Buying a new laptop can be a difficult venture. You must decide which one is right for you. Depending on what your needs are, there are tons of things to consider like hard drive space, graphics cards, and general ease of use.</p> 
       <p>But once you find your dream computer, there are a few things that you must do. Here are some tips to follow after you purchase your laptop to make your computing experience a pleasurable one.</p> 
       <h2>Register and Update Windows</h2> 
       <p><a href="http://blog.superuser.com/files/2013/10/win8.jpg"><img class="alignnone size-full wp-image-5527" alt="win8" src="http://blog.superuser.com/files/2013/10/win8.jpg" width="500" height="333"></a></p> 
       <p>Image via <a href="http://www.flickr.com/photos/46411239@N05/8121841101/in/photolist-dnGyp8-dnHY2R-dnHG8k-d2HaGf-d2H6Fs-d2H8Ry-d2H8CW-d2Hb4u-bmCLbh-doAybA-dTaUm9-eguBC6-brXLxq-bESDEM-bESDux-bzphKr-apQp58-apQp56-bVmug4-bAL7w3-dnU4xL-92andR-doAxzQ-doAtnn-cz65YA-doAri2-doApue-doABxd-doAvTq-dovQmo-dovGSp-dovCvv-dcvNUn-bPEM8T-amWZJK-amX12v-cqpQ21-dihuNQ-f7Da7N-f7oYoX-f7oYCD-dP5mLe-dGi7eA-dorutS-dnDuwT-em732U-dsSyjE-bnhnaL-ciT1ZE-e6UyZ8-brXLU5" target="_blank">Flickr</a> by Microsoft Sweden</p> 
       <p>An important part of purchasing your computer is actually registering (and <a href="http://windows.microsoft.com/en-us/windows-8/why-activate-windows">successfully activating</a>) Windows. It activates all the perks of having Windows as an operating system, such as Windows Media Player, and it also enables desktop personalization.</p> 
       <p>Next you’ll want to download all system updates and service packs. You’ll want a really fast <a href="http://www.dsl.com/tools/internet-cable-breakdown/">internet connection</a> for this, because these can be huge files and take a while to download. However, they’re vital to making your computer safer and run much smoother. These updates patch up any bugs or glitches that were newly found, and they streamline the performance of the operating system as well as add new features. &nbsp;To <a href="http://superuser.com/questions/545115/is-windows-7-home-better-for-small-notebook-than-professional">answer super user “Hennes” question</a>, it doesn’t matter which variation of Windows 7 you have, Pro or Home, it will run, performance wise, the same, if you’re worried about performance issues think about either adding more RAM or upgrading to Windows 8, which for the for the most part is a more streamlined and smoother operating system.</p> 
       <h2>Rid your Computer from Unwanted “Bloatware”</h2> 
       <p>When you purchase a computer, you’d imagine that you’re starting with a clean slate. However, this couldn’t be further from the truth. Most computers are loaded with unwanted games and software (referred to as bloatware, since it bulks up what should be a clean slate). The most efficient way to truly free up your computer is to install a brand-new, store-bought copy of Windows onto your hard drive.&nbsp; This will wipe out everything that was on the hard drive (so make sure you save everything you wanted to keep on an external drive or disc) and leave a clean and smoothly operating system, free of unwanted, useless bloatware.</p> 
       <p>If you don’t feel like doing this, or don’t want to buy a new copy of Windows, you can try manually uninstalling the offending software, by going to Start &gt; Control Panel &gt; Programs and Features. From here you can go to each program you don’t want and click uninstall. For programs that are more deeply rooted, like anti-virus software, you can go to the developers website and search for the “complete uninstall” procedure and follow their steps to completely eradicate it from your system.</p> 
       <h2>Anti-virus and Security Software</h2> 
       <p>While many computers come with standard anti-virus software, it’s important to make sure you have the right program to deal with potential intruders. That being said, you may want to consider forking out a few dollars for an all-encompassing anti-virus program, such as Norton Antivirus software, or save some money and get basic protection, with something like <a href="http://free.avg.com/us-en/homepage">AVG Anti-Virus</a> software.</p> 
       <p>Going together with anti-virus software is security scanning software. What you’re looking for are programs such as malware that will make your computer run at a turtle’s pace. By running a security scan, you can remove all the unwanted unintentionally downloaded programs off your hard drive before you get into computing that would release sensitive information.</p> 
       <p>Schedule a security scan for about once a week (you can set it to automatically begin when you want, in whatever increment you choose) to keep your computer free from malicious programs. Keep this software running in the background, and it will detect and quarantine any questionable and infected files that you may have just unintentionally downloaded and it will also warn you and deter you away from possibly unsafe sites.</p> 
       <h2>Back Up Software and Recovery</h2> 
       <p>Any computer with Windows will have system recovery loaded to it. Recovery restores your computer to a pre-existing state if the worst should happen.</p> 
       <p>If you drop your computer or it suddenly fails due to a power surge or something else, recovery will make your computer work again if possible. Backing up, on the other hand, is basically insurance for your computer. You can take all the files that you never want to lose, and you can put them on an external hard drive. Or if you don’t want to buy one of those, you can copy the files on to a DVD or CD and keep them in a safe place. These measures ensure that you will never have to fret over lost work.</p> 
       <p>Physical back-ups aren’t your only option – there are plenty of ways to <a href="http://www.techradar.com/us/news/world-of-tech/roundup/how-to-choose-a-cloud-back-up-service-1087006">back up your info using an online cloud service</a>, doing this will automatically back up your files as you make them, and no matter what happens to your computer, or backup drives, your files will be downloadable from their hard drives on their servers.</p> 
       <h2>Power Saving</h2> 
       <p>Make sure that, after registering Windows, you go into your personalization and check your power saving scheme. Here, you can <a href="http://www.thewindowsclub.com/configure-power-plan-windows-8">configure your Windows 8 power plan settings and options</a>. Choose how long until your computer turns off after it’s idle, as well as setting a screen saver. Both of these things will save you on your energy bill and keep your computer running longer.</p> 
       <p>No matter what laptop you choose to buy, following a few simple steps for your new laptop will go a long way. Although you shouldn’t expect problems on a brand new machine, it’s all about peace of mind. You can use the computer at your leisure and not be worried about the problems that can compound over time.</p>

          <p>I have to contest your security software advice. I’ve often found that it is in fact Norton and AVG that “make your computer run at a turtle’s pace”. I’m not sure what the best course of action is, though, but I can think of three guidelines:</p> 
          <ol> 
           <li><p><strong>In your web browser, enable click-to-play for all plug-ins.</strong> If you want, you can whitelist sites with an outstanding reputation for security, like YouTube.</p> <p>In many web browser-based attacks, usually one of these three pieces of software are part of the problem: Adobe’s Flash plug-in, Adobe’s PDF Reader plug-in, and Oracle’s Java applet plug-in. They are often used in so-called ‘drive-by attacks’.</p> <p>To further shield yourself from these attacks, you can take the following additional steps:</p> <p>a. Uninstall Adobe Reader. Get other software to read PDF files, like Foxit Reader or Sumatra. b. Disable or even uninstall the Java applet plug-in. It’s distinct from Java Runtime Environment, which (sadly) you probably still need for a bunch of software.</p></li> 
           <li><p><strong>Do not run untrusted software.</strong> Did you receive a file through e-mail or chat? Good for you. Did the file name end in exe or com? Don’t run it. If you don’t know whether to trust software or not, assume it’s <em>untrusted</em>.</p> <p>Also, tell Windows to <a href="http://windows.microsoft.com/en-us/windows/show-hide-file-name-extensions#show-hide-file-name-extensions=windows-vista" rel="nofollow">not hide known extensions</a>.</p></li> 
           <li><p><strong>Always run updates.</strong> Seriously. Run Windows updates. Let Java update itself. Update iTunes. It doesn’t matter how long it takes, just do it.</p></li>
