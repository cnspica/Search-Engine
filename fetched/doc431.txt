        <p>Entity framework is an Object Relational Mapping (ORM) framework that offers an automated mechanism to developers for storing and accessing the data in the database. This tutorial covers the features of Entity Framework using Code First approach. It also explains the new features introduced in Entity Framework 6.</p> 
        <h1>Audience</h1> 
        <p>This tutorial is designed for those who want to learn how to start the development of Entity Framework in their application.</p> 
        <h1>Prerequisites</h1> 
        <p>You should have a basic knowledge of Visual Studio, C#, and MS SQL Server to make the most of this tutorial.</p>
