        <p>CoffeeScript is a light weight language which transcompiles into JavaScript. It provides better syntax avoiding the quirky parts of JavaScript, still retaining the flexibility and beauty of the language.</p> 
        <h1>Audience</h1> 
        <p>This tutorial has been prepared for beginners to help them understand the basic functionality of CoffeeScript to build dynamic webpages and web applications.</p> 
        <h1>Prerequisites</h1> 
        <p>For this tutorial, it is assumed that the readers have a prior knowledge of HTML coding and JavaScript. It would help if the reader has some prior exposure to object-oriented programming concepts and a general idea on creating online applications.</p> 
        <h1>Execute CoffeeScript Online</h1> 
        <p>For most of the examples given in this tutorial, you will find <b>Try it</b> option, so just make use of this option to transcompile your CoffeeScript programs to JavaScript programs on the spot and enjoy your learning.</p> 
        <p>Try the following example using the <b>Try it</b> option available at the top right corner of the below sample code box −</p>
