        <p>Spark is Hadoop’s sub-project. Therefore, it is better to install Spark into a Linux based system. The following steps show how to install Apache Spark.</p> 
        <h2>Step 1: Verifying Java Installation</h2> 
        <p>Java installation is one of the mandatory things in installing Spark. Try the following command to verify the JAVA version.</p> 
        <pre class="result notranslate">
$java -version 
</pre> 
        <p>If Java is already, installed on your system, you get to see the following response −</p> 
        <pre class="result notranslate">
java version "1.7.0_71" 
Java(TM) SE Runtime Environment (build 1.7.0_71-b13) 
Java HotSpot(TM) Client VM (build 25.0-b02, mixed mode)
</pre> 
        <p>In case you do not have Java installed on your system, then Install Java before proceeding to next step.</p> 
        <h2>Step 2: Verifying Scala installation</h2> 
        <p>You should Scala language to implement Spark. So let us verify Scala installation using following command.</p> 
        <pre class="result notranslate">
$scala -version
</pre> 
        <p>If Scala is already installed on your system, you get to see the following response −</p> 
        <pre class="result notranslate">
Scala code runner version 2.11.6 -- Copyright 2002-2013, LAMP/EPFL
</pre> 
        <p>In case you don’t have Scala installed on your system, then proceed to next step for Scala installation.</p> 
        <h2>Step 3: Downloading Scala</h2> 
        <p>Download the latest version of Scala by visit the following link <a target="_blank" rel="nofollow" href="http://www.scala-lang.org/download/">Download Scala</a>. For this tutorial, we are using scala-2.11.6 version. After downloading, you will find the Scala tar file in the download folder.</p> 
        <h2>Step 4: Installing Scala</h2> 
        <p>Follow the below given steps for installing Scala.</p> 
        <h3>Extract the Scala tar file</h3> 
        <p>Type the following command for extracting the Scala tar file.</p> 
        <pre class="result notranslate">
$ tar xvf scala-2.11.6.tgz
</pre> 
        <h3>Move Scala software files</h3> 
        <p>Use the following commands for moving the Scala software files, to respective directory <b>(/usr/local/scala)</b>.</p> 
        <pre class="result notranslate">
$ su – 
Password: 
# cd /home/Hadoop/Downloads/ 
# mv scala-2.11.6 /usr/local/scala 
# exit 
</pre> 
        <h3>Set PATH for Scala</h3> 
        <p>Use the following command for setting PATH for Scala.</p> 
        <pre class="result notranslate">
$ export PATH = $PATH:/usr/local/scala/bin
</pre> 
        <h3>Verifying Scala Installation</h3> 
        <p>After installation, it is better to verify it. Use the following command for verifying Scala installation.</p> 
        <pre class="result notranslate">
$scala -version
</pre> 
        <p>If Scala is already installed on your system, you get to see the following response −</p> 
        <pre class="result notranslate">
Scala code runner version 2.11.6 -- Copyright 2002-2013, LAMP/EPFL
</pre> 
        <h2>Step 5: Downloading Apache Spark</h2> 
        <p>Download the latest version of Spark by visiting the following link <a target="_blank" rel="nofollow" href="https://spark.apache.org/downloads.html">Download Spark</a>. For this tutorial, we are using <b>spark-1.3.1-bin-hadoop2.6</b> version. After downloading it, you will find the Spark tar file in the download folder.</p> 
        <h2>Step 6: Installing Spark</h2> 
        <p>Follow the steps given below for installing Spark.</p> 
        <h3>Extracting Spark tar</h3> 
        <p>The following command for extracting the spark tar file.</p> 
        <pre class="result notranslate">
$ tar xvf spark-1.3.1-bin-hadoop2.6.tgz 
</pre> 
        <h3>Moving Spark software files</h3> 
        <p>The following commands for moving the Spark software files to respective directory <b>(/usr/local/spark)</b>.</p> 
        <pre class="result notranslate">
$ su – 
Password:  

# cd /home/Hadoop/Downloads/ 
# mv spark-1.3.1-bin-hadoop2.6 /usr/local/spark 
# exit 
</pre> 
        <h3>Setting up the environment for Spark</h3> 
        <p>Add the following line to ~<b>/.bashrc</b> file. It means adding the location, where the spark software file are located to the PATH variable.</p> 
        <pre class="result notranslate">
export PATH = $PATH:/usr/local/spark/bin
</pre> 
        <p>Use the following command for sourcing the ~/.bashrc file.</p> 
        <pre class="result notranslate">
$ source ~/.bashrc
</pre> 
        <h2>Step 7: Verifying the Spark Installation</h2> 
        <p>Write the following command for opening Spark shell.</p> 
        <pre class="result notranslate">
$spark-shell
</pre> 
        <p>If spark is installed successfully then you will find the following output.</p> 
        <pre class="result notranslate">
Spark assembly has been built with Hive, including Datanucleus jars on classpath 
Using Spark's default log4j profile: org/apache/spark/log4j-defaults.properties
