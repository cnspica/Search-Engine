           <td><a href="/nodejs/nodejs_net_module.htm" alt="Net Module"><b>Net Module</b></a><br> Provides both servers and clients as streams. Acts as a network wrapper.</td>

           <td><a href="/nodejs/nodejs_dns_module.htm" alt="DNS Module"><b>DNS Module</b></a><br> Provides functions to do actual DNS lookup as well as to use underlying operating system name resolution functionalities.</td>
