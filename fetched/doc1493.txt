        <p>Cucumber is a testing tool that supports Behavior Driven Development (BDD) framework. It defines application behavior using simple English text, defined by a language called Gherkin.</p> 
        <p>Cucumber allows automation functional validation that is easily read and understood. Cucumber was initially implemented in Ruby and then extended to Java framework. Both the tools support native JUnit.</p> 
        <p>This tutorial is fairly comprehensive and covers all the necessary aspects on Cucumber using examples for easy understanding.</p> 
        <h1>Audience</h1> 
        <p>This tutorial is designed for software professionals such as analysts, developers, and testers who are keen on learning the fundamentals of Cucumber and want to put it into practice.</p> 
        <h1>Prerequisites</h1> 
        <p>Before proceeding with this tutorial, you need to have a basic knowledge on testing as well as some hands-on experience of some testing tools. You should have a commanding knowledge on Java, and some familiarity with JUnit and Ruby.</p>
