        <hr> 
        <p>Dear readers, these <b>Joomla Interview Questions</b> have been designed specially to get you acquainted with the nature of questions you may encounter during your interview for the subject of <b>Joomla</b>. As per my experience good interviewers hardly plan to ask any particular question during your interview, normally questions start with some basic concept of the subject and later they continue based on further discussion and what you answer:</p> 
        <section class="toggle"> 
         <label>What is Joomla?</label> 
         <div class="toggle-content"> 
          <p>Joomla is an open source Content Management System (CMS), which is used to build websites and online applications. It is free and extendable which is separated into front-end and back-end (administrator). </p> 
         </div> 
        </section> 
        <section class="toggle"> 
         <label>List some features of Joomla?</label> 
         <div class="toggle-content"> 
          <p>The features of Joomla are:</p> 
          <ul class="list"> 
           <li><p>User Manager</p></li> 
           <li><p>Content Manager</p></li> 
           <li><p>Banner Manager</p></li> 
           <li><p>Template Manager</p></li> 
           <li><p>Media Manager</p></li> 
           <li><p>Contact Manager</p></li> 
           <li><p>Search</p></li> 
          </ul> 
         </div> 
        </section> 
        <section class="toggle"> 
         <label>What is Joomla Framework? </label> 
         <div class="toggle-content"> 
          <p>Framework is collection of open source software, where the Joomla CMS is built on. It developed for more flexibility and breaks the framework into single modular packages which helps each package to develop more easily.</p> 
         </div> 
        </section> 
        <section class="toggle"> 
         <label>What database does Joomla supports?</label> 
         <div class="toggle-content"> 
          <p>MySql Database</p> 
         </div> 
        </section> 
        <section class="toggle"> 
         <label>What are disadvantages of Joomla?</label> 
         <div class="toggle-content"> 
          <p>The disadvantages of Joomla are:</p> 
          <ul class="list"> 
           <li><p>It gives compatibility problem while installing several modules, extensions and plugins simultaneously.</p></li> 
           <li><p>Plugins and modules are not free in Joomla.</p></li> 
           <li><p>Development is too difficult to handle when you want to change the layout.</p></li> 
           <li><p>Joomla is not much SEO (Search Engine Optimization) friendly.</p></li> 
           <li><p>It makes website heavy to load and run.</p></li> 
          </ul> 
         </div> 
        </section> 
        <section class="toggle"> 
         <label>What is Joomla Components?</label> 
         <div class="toggle-content"> 
          <p>Components are considered as mini applications. It consists of two parts i.e. Administrator and Site. Whenever a page gets loads, component is been called to render the body of main page. The Administrator part manages different aspects of the component and the site part helps in rendering the pages when request is made by site visitor. Components are main functional unit of joomla.</p> 
         </div> 
        </section> 
        <section class="toggle"> 
         <label>What is Joomla Modules?</label> 
         <div class="toggle-content"> 
          <p>Modules are an extension which is used to render the pages in Joomla. It is also used to display the new data from the component. It frequently looks like boxes such as login module. In Joomla administrator the modules are managed by the module manager. It displays the new content and images when module is linked to joomla components.</p> 
         </div> 
        </section> 
        <section class="toggle"> 
         <label>What is Joomla Plugins?</label> 
         <div class="toggle-content"> 
          <p>This is also a kind of Joomla extension; it is a very flexible and powerful for extending the framework. It contain a bit of codes that is used to execute the particular event trigger, it is commonly used to format the output of a component or module when page is built. The plugin functions which are associated with event are executed in a sequence when a particular event occurs.</p> 
         </div> 
        </section> 
        <section class="toggle"> 
         <label>What is the use of Breadcrumb Module in Joomla? </label> 
         <div class="toggle-content"> 
          <p>Breadcrumb is used to display the hierarchical representation of the navigation links so that the users can easily understand where he/she is in the website and allow them to navigate back.</p> 
         </div> 
        </section> 
        <section class="toggle"> 
         <label>What does Check In mean is article manager toolbar? </label> 
         <div class="toggle-content"> 
          <p>It check-in the particular selected article. Multiple articles can be checked-in.</p> 
         </div> 
        </section> 
        <section class="toggle"> 
         <label>What is the use of Meta Description in Joomla Menu Manager? </label> 
         <div class="toggle-content"> 
          <p>It is used as an optional paragraph which includes a description of the page and is displayed in the results of the search engines.</p> 
         </div> 
        </section> 
        <section class="toggle"> 
         <label>What is the use of Secure in Joomla Menu Manager?</label> 
         <div class="toggle-content"> 
          <p>It defines whether link must use SSL or Secure Site URL. It provides three options such as Off, On and Ignore.</p> 
         </div> 
        </section> 
        <section class="toggle"> 
         <label>What is does <i>Hits</i> column mean in article manager page?</label> 
         <div class="toggle-content"> 
          <p>It specifies the number of the times the item was viewed.</p> 
         </div> 
        </section> 
        <section class="toggle"> 
         <label>Which editor is used in Joomla?</label> 
         <div class="toggle-content"> 
          <p>TinyMCE Editor.</p> 
         </div> 
        </section> 
        <section class="toggle"> 
         <label>Which are the Editor buttons in TinyMCE Editor? </label> 
         <div class="toggle-content"> 
          <p>The Editor buttons are:</p> 
          <ul class="list"> 
           <li><p>Article</p></li> 
           <li><p>Image</p></li> 
           <li><p>Page Break</p></li> 
           <li><p>Read More</p></li> 
           <li><p>Toggle Editor.</p></li> 
          </ul> 
         </div> 
        </section> 
        <section class="toggle"> 
         <label>What does the Purchase Type column indicates in Banner Manager?</label> 
         <div class="toggle-content"> 
          <p>It indicates how the clients are purchasing the banners i.e. yearly or Monthly. By default it is monthly.</p> 
         </div> 
        </section> 
        <section class="toggle"> 
         <label>What is the use of Extension Manager in Joomla?</label> 
         <div class="toggle-content"> 
          <p>Extension Manager is used to extend the functionality of Joomla website.</p> 
         </div> 
        </section> 
        <section class="toggle"> 
         <label>What is the use Official Support Forum in Joomla?</label> 
         <div class="toggle-content"> 
          <p>Official Support Forum helps in supporting the exchange of ideas and views officially in Joomla.</p> 
         </div> 
        </section> 
        <section class="toggle"> 
         <label>What is the use of Documentation Wiki in Joomla?</label> 
         <div class="toggle-content"> 
          <p>The Documentation Wiki is used for formatting the content of articles, cross linking pages and creating the documentation of Wiki template by themselves.</p> 
         </div> 
        </section> 
        <section class="toggle"> 
         <label>What is Joomla Translations?</label> 
         <div class="toggle-content"> 
          <p>Joomla translations help in translation of language in Joomla core files. It helps in documentation and screens along with other task to meet the need of diverse language community.</p> 
         </div> 
        </section> 
        <section class="toggle"> 
         <label>What is the use of Module Class Suffix in Breadcrumb Module?</label> 
         <div class="toggle-content"> 
          <p>Setting this parameter causes Joomla to either add a new CSS class or modify the existing CSS class for the div element for this specific module.</p> 
         </div> 
        </section> 
        <section class="toggle"> 
         <label>What is Feed Display Module in Joomla?</label> 
         <div class="toggle-content"> 
          <p>Feed Display Module is used to show the RSS News Feeds from a website. It is a way to provide user with the updated content from other resources.</p> 
         </div> 
        </section> 
        <section class="toggle"> 
         <label>What is Footer Module?</label> 
         <div class="toggle-content"> 
          <p>Footer displays the information of the Joomla license and the copyright of the website.</p> 
         </div> 
        </section> 
        <section class="toggle"> 
         <label>What is Latest News Module?</label> 
         <div class="toggle-content"> 
          <p>Latest News Module is used to display the most recent published article in a list.</p> 
         </div> 
        </section> 
        <section class="toggle"> 
         <label>What is Search Module?</label> 
         <div class="toggle-content"> 
          <p>Search Module is used to display a search box, where the user types a particular name to search the website.</p> 
         </div> 
        </section> 
        <section class="toggle"> 
         <label>What is Random Image Module?</label> 
         <div class="toggle-content"> 
          <p>Random Image Module is used to display the images randomly from the selected directory.</p> 
         </div> 
        </section> 
        <section class="toggle"> 
         <label>What is Who's Online Module?</label> 
         <div class="toggle-content"> 
          <p>This module helps to display the user's information that is accessing the website.</p> 
         </div> 
        </section> 
        <section class="toggle"> 
         <label>What is Syndicate Module?</label> 
         <div class="toggle-content"> 
          <p>Syndicate module helps in creating an RSS Feed link for the page. It allows creating a Newsfeed by user for the current page.</p> 
         </div> 
        </section> 
        <section class="toggle"> 
         <label>What is the use of Path to Log Folder under system setting?</label> 
         <div class="toggle-content"> 
          <p>It gives a path to store the logs by Joomla. This path is automatically set up on Joomla installation and should not be changed.</p> 
         </div> 
        </section> 
        <section class="toggle"> 
         <label>What is the use of Joomla Media Settings?</label> 
         <div class="toggle-content"> 
          <p>Joomla Media Settings help to configure the global media files options such like file format, MIME specifications, upload of files, size of files etc.</p> 
         </div> 
        </section> 
        <section class="toggle"> 
         <label>What is Language Manager?</label> 
         <div class="toggle-content"> 
          <p>The Joomla language manager option is used to set the Native Title, Language Code, SEF Prefix, and Image Prefixes of the installed or to be installed languages.</p> 
         </div> 
        </section> 
        <section class="toggle"> 
         <label>What is the use of Mass Emailing in Joomla?</label> 
         <div class="toggle-content"> 
          <p>Mass Mail is used for sending emails to the group of registered users. Users can be selected based on groups.</p> 
         </div> 
        </section> 
        <section class="toggle"> 
         <label>What is the use of Debug system section under debug setting?</label> 
         <div class="toggle-content"> 
          <p>It is a debugging system of Joomla which provides debug information by setting it to "Yes". It provides some various forms such as diagnostic information, language translations and SQL errors. </p> 
         </div> 
        </section> 
        <section class="toggle"> 
         <label>What is Template Manager?</label> 
         <div class="toggle-content"> 
          <p>It manages the various templates that are used in the website. The templates can be used without changing the content structure of the website.</p> 
         </div> 
        </section>
