        <p><b>Meteor</b> is full-stack JavaScript platform for building web and mobile apps. It makes it easier to create real time apps since Meteor alone offers full ecosystem to work with instead of combining couple of different tools and framework to get the same effect.</p> 
        <h1>Audience</h1> 
        <p>This tutorial will help any JavaScript developer who wants to learn Meteor framework. We will try to show you simple, easy to grasp code samples and crisp explanation. Beginners will benefit the most from these tutorials but you can also use it as a reference when working with Meteor framework.</p> 
        <h1>Prerequisites</h1> 
        <p>Meteor is JavaScript framework so you need to have basic knowledge of <b>JavaScript</b> and <b>HTML</b>. You will also need to be familiar with NodeJS and MongoDB, although you will be able to understand everything without previous knowledge. If you never used mongoDB, any knowledge about databases should suffice. Since Meteor is full-stack framework, any previous experience in creating web or mobile apps will be helpful.</p>
