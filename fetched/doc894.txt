        <p>Joomla is an open source Content Management System (CMS), which is used to build websites and online applications. It is free and extendable which is separated into front-end templates and back-end templates (administrator). Joomla is developed using PHP, Object Oriented Programming, software design patterns and MySQL (used for storing the data). This tutorial will teach you the basics of Joomla using which you can create websites with ease. The tutorial is divided into sections such as Joomla Basics, Joomla Menus, Joomla Modules, Joomla Global Settings, and Joomla Advanced. Each of these sections contain related topics with screenshots explaining the Joomla admin screens.</p>
