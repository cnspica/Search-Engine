          <h2 class="user-card-name"> balpha<span class="mod-flair" title="moderator">♦</span> <span>(moderator)</span> <span class="top-badge"></span> <script> $(function() { $(".top-badge").addSpinner().load("/users/rank?userId=115866"); }); </script> </h2> 
          <div class="bio"> 
           <p>My name is Benjamin Dumke-von der Ehe. I am <a href="http://blog.stackoverflow.com/2010/06/welcome-stack-overflow-valued-associates-00006-and-00007">Stack Overflow Valued Associate #00007</a>, which is just a confusing way of saying that I work as a developer for Stack Overflow.</p> 
           <p>I am <a href="http://twitter.com/balpha" rel="nofollow">@balpha</a> on Twitter. My <a href="http://balpha.de/about/" rel="nofollow">blog's "about" page</a> explains the origin of my user name.</p> 
           <p>I am the creator of <a href="http://unicornify.appspot.com/" rel="nofollow"><em>Unicornify</em></a>, the service that gives you these lovable creatures:</p> 
           <p><img src="http://unicornify.appspot.com/avatar/1?s=48" alt=""> <img src="http://unicornify.appspot.com/avatar/13?s=48" alt=""> <img src="http://unicornify.appspot.com/avatar/3?s=48" alt=""> <img src="http://unicornify.appspot.com/avatar/14?s=48" alt=""> <img src="http://unicornify.appspot.com/avatar/5?s=48" alt=""></p> 
           <p>I also created <a href="https://bitbucket.org/balpha/lyfe" rel="nofollow"><em>Lyfe</em></a>, a small library adding <code>yield</code>-based generators to JavaScript, and <a href="http://notarealquestion.appspot.com/" rel="nofollow"><em>Not a Real Question</em></a>, a little game that's played with data from various Stack Exchange sites.</p> 
           <p>I sometimes have a <a href="http://meta.stackexchange.com/questions/43418">strange</a> <a href="http://stackoverflow.com/questions/2057400">sense of</a> <a href="http://meta.stackexchange.com/questions/122781">humor</a>, but I also like to <a href="http://meta.stackexchange.com/a/74115">explain</a> <a href="http://stackoverflow.com/a/2055144">things</a> <a href="http://meta.stackexchange.com/a/75045">I fix</a>, <a href="http://meta.stackexchange.com/a/128705">break</a>, <a href="http://balpha.de/2012/07/an-unexcited-look-at-browser-sniffing" rel="nofollow">care about</a>, or <a href="http://stackoverflow.com/a/11833858">come up with</a>.</p>
