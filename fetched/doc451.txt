        <p>JSON file stores data as text in human-readable format. Json stands for JavaScript Object Notation. R can read JSON files using the rjson package.</p> 
        <h2>Install rjson Package</h2> 
        <p>In the R console, you can issue the following command to install the rjson package.</p> 
        <pre class="result notranslate">
install.packages("rjson")
</pre> 
        <h2>Input Data</h2> 
        <p>Create a JSON file by copying the below data into a text editor like notepad. Save the file with a <b>.json</b> extension and choosing the file type as <b>all files(*.*)</b>.</p> 
        <pre class="prettyprint notranslate" id="input_json">
{ 
   "ID":["1","2","3","4","5","6","7","8" ],
   "Name":["Rick","Dan","Michelle","Ryan","Gary","Nina","Simon","Guru" ],
   "Salary":["623.3","515.2","611","729","843.25","578","632.8","722.5" ],
   
   "StartDate":[ "1/1/2012","9/23/2013","11/15/2014","5/11/2014","3/27/2015","5/21/2013",
      "7/30/2013","6/17/2014"],
   "Dept":[ "IT","Operations","IT","HR","Finance","IT","Operations","Finance"]
}
</pre> 
        <h2>Read the JSON File</h2> 
        <p>The JSON file is read by R using the function from <b>JSON()</b>. It is stored as a list in R.</p> 
        <pre class="prettyprint notranslate tryit" title=",,,input_json">
# Load the package required to read JSON files.
library("rjson")

# Give the input file name to the function.
result &lt;- fromJSON(file = "input.json")

# Print the result.
print(result)
</pre> 
        <p>When we execute the above code, it produces the following result −</p> 
        <pre class="result notranslate">
$ID
[1] "1"   "2"   "3"   "4"   "5"   "6"   "7"   "8"

$Name
[1] "Rick"     "Dan"      "Michelle" "Ryan"     "Gary"     "Nina"     "Simon"    "Guru"

$Salary
[1] "623.3"  "515.2"  "611"    "729"    "843.25" "578"    "632.8"  "722.5"

$StartDate
[1] "1/1/2012"   "9/23/2013"  "11/15/2014" "5/11/2014"  "3/27/2015"  "5/21/2013"
   "7/30/2013"  "6/17/2014"

$Dept
[1] "IT"         "Operations" "IT"         "HR"         "Finance"    "IT"
   "Operations" "Finance"
</pre> 
        <h2>Convert JSON to a Data Frame</h2> 
        <p>We can convert the extracted data above to a R data frame for further analysis using the <b>as.data.frame()</b> function.</p> 
        <pre class="prettyprint notranslate tryit" title=",,,input_json">
# Load the package required to read JSON files.
library("rjson")

# Give the input file name to the function.
result &lt;- fromJSON(file = "input.json")

# Convert JSON file to a data frame.
json_data_frame &lt;- as.data.frame(result)

print(json_data_frame)
</pre> 
        <p>When we execute the above code, it produces the following result −</p> 
        <pre class="result notranslate">
      id,   name,    salary,   start_date,     dept
1      1    Rick     623.30    2012-01-01      IT

5     NA    Gary     843.25    2015-03-27      Finance
6      6    Nina     578.00    2013-05-21      IT
7      7    Simon    632.80    2013-07-30      Operations
8      8    Guru     722.50    2014-06-17      Finance
</pre>
